2.3 Problema modelo

Nesta seção, apresentaremos a aplicação do método de elementos finitos para a equação de Poisson11 1 Siméon Denis Poisson, 1781 - 1840, matemático francês. Fonte: Wikipedia. com condições de Dirichlet22 2 Johann Peter Gustav Lejeune Dirichlet, 1805 - 1859, matemático alemão. Fonte: Wikipedia., i.e.: encontrar u tal que

-Δ⁢u =f,x∈Ω, (2.24)
u =0,x∈∂⁡Ω, (2.25)

onde Δ=∂2/∂⁡x02+∂2/∂⁡x12 é o operador de Laplace33 3 Pierre-Simon, marquis de Laplace, 1749 - 1827, matemático francês. Fonte: Wikipedia. e f é uma função dada.

2.3.1 Formulação variacional

A aplicação do método de elementos finitos é construída sobre a formulação fraca do problema (2.24)-(2.25). Para obtermos esta, multiplicamos (2.24) por uma função teste v em um espaço adequado V0 e integramos no domínio Ω, i.e.

-∫ΩΔ⁢u⁢v⁢𝑑x=∫Ωf⁢v⁢𝑑x. (2.26)

Então, usando a fórmula de Green44 4 George Green, 1793 - 1841, matemático britânico. Fonte: Wikipedia., obtemos

∫Ω∇⁡u⋅∇⁡v⁢d⁢x-∫∂⁡Ωn⋅∇⁡u⁢v⁢d⁢s. (2.27)

Então, observando critérios de regularidade e a condição de contorno (2.25), escolhemos

V0:={v∈H1⁢(Ω):v|∂⁡Ω=0}. (2.28)

Lembramos que H1⁢(Ω)={v:∥v∥L2⁢(Ω)+∥∇⁡v∥L2⁢(Ω)<∞}.

Com isso, temos o seguinte problema fraco associado a (2.24)-(2.25): encontrar u∈V0 tal que

a⁢(u,v)=L⁢(v),∀v∈V0, (2.29)

onde a⁢(u,v) é chamada de forma bilinear e definida por

a⁢(u,v):=∫Ω∇⁡u⋅∇⁡v⁢d⁢x (2.30)

e L⁢(v) é chamada de forma linear e definida por

L⁢(v):=∫Ωf⁢v⁢𝑑x. (2.31)

2.3.2 Formulação de elementos finitos

A formulação de elementos finitos é obtida da formulação fraca (2.29) pela aproximação do espaço teste V0 por uma espaço de dimensão finita. Tomando uma triangulação 𝒦⊂Ω e considerando o espaço contínuo dos polinômios lineares por partes

Vh:={v:v∈C0⁢(Ω),v|K∈P1⁢(K)⁢∀K∈𝒦}, (2.32)

assumimos também o subconjunto Vh⁢,0:={v∈Vh:v|∂⁡Ω=0}.

Com isso, temos o seguinte problema de elementos finitos associado (2.29): encontrar uh∈Vh⁢,0 tal que

a⁢(uh,vh)=L⁢(vh),∀vh∈Vh⁢,0. (2.33)

Observemos que (2.33) é equivalente ao problema de encontrar uh∈Vh⁢,0 tal que

a⁢(uh,φi)=L⁢(φi), (2.34)

com i=0,1,⋯,np-1, onde {φi}i=0ni-1 é a base nodal de Vh⁢,0 e ni é o número de funções bases (igual ao número de nodos internos da triangulação 𝒦). Ainda, como

uh=∑j=0ni-1ξj⁢φj, (2.35)

temos

a⁢(uh,φi) =a⁢(∑j=0ni-1ξj⁢φj,φi) (2.36)
=∑j=0ni-1ξj⁢a⁢(φj,φi). (2.37)

Com isso, o problema de elementos finitos é equivalente a resolver o seguinte sistema linear

∑j=0ni-1ξj⁢a⁢(φj,φi)=L⁢(φi),i=0,1,⋯,ni-1, (2.38)

para as incógnitas ξj, j=0,1,⋯,ni-1. Ou, equivalentemente, temos sua forma matricial

A⁢𝝃=𝒃, (2.39)

onde A=[ai,j]i,j=0ni-1 é chamada de matriz de rigidez com

ai,j=a⁢(φj,φi) (2.40)

e 𝒃=(b0,b1,⋯,bni-1) é o vetor de carga com

bi=L⁢(φi). (2.41)
Exemplo 2.3.1.

Consideremos o seguinte problema de Poisson

-Δ⁢u =100⁢x0⁢(1-x0)⁢x1⁢(1-x1),x∈Ω:=(0,1)×(0,1), (2.42)
u =0,x∈∂⁡Ω. (2.43)

Na Figura 2.2 temos um esboço da aproximação de elementos finitos obtida em uma malha uniforme com 20×20 nodos. As isolinhas correspondem aos ponto tais que u=3×10-1⁢,2×10-1, 10-1, 5×10-2.

Com o FEniCS, podemos computar a solução deste problema com o seguinte código:

from __future__ import print_function, division
from fenics import *
import numpy as np
import matplotlib.pyplot as plt

# malha
Nx = 20
Ny = 20
mesh = UnitSquareMesh(Nx,Ny)

# espaco
V = FunctionSpace(mesh, ’P’, 1)

# cond. contorno
def boundary(x,on_boundary):
    return on_boundary

bc = DirichletBC(V,Constant(0.0),boundary)

# f
f = Expression(’100*x[0]*(1-x[0])*x[1]*(1-x[1])’,degree=4)

# MEF problem
u = TrialFunction(V)
v = TestFunction(V)
a = dot(grad(u), grad(v))*dx
L = f*v*dx

#computa a sol
u = Function(V)
solve(a == L, u, bc)

# exportanto em vtk
vtkfile = File(’u.pvd’)
vtkfile << u
