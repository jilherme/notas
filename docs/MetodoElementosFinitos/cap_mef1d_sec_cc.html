1.3 Condições de contorno

Nesta seção, vamos discutir sobre soluções de elementos finitos para a equações diferencial

-u′′=f,x∈I=[0,L], (1.102)

com diferentes condições de contorno.

1.3.1 Condições de Dirichlet

Consideremos o seguinte problema com condições de contorno de Dirichlet44Johann Peter Gustav Lejeune Dirichlet, 1805 - 1859, matemático alemão. Fonte: Wikipedia.: encontrar u tal que

-u′′=f,∀x∈I=[0,L], (1.103)
u⁢(0)=u0,u⁢(L)=uL, (1.104)

com u0, uL e f dados.

Tomando uma função teste v∈V0=H01(I):={v∈H1(I);v(0)=v(L)=0} e multiplicando-a em (1.103), obtemos

-∫Iu′′⁢v⁢𝑑x=∫If⁢v⁢𝑑x. (1.105)

Aplicando a integração por partes, temos

∫Iu′⁢v′⁢𝑑x=∫If⁢v⁢𝑑x. (1.106)

Desta forma, definimos o seguinte problema fraco associado: encontrar u∈V:={v∈H1(I);v(0)=u0,v(L)=vL} tal que

a⁢(u,v)=L⁢(v),∀v∈V0, (1.107)

onde a⁢(u,v) é a forma bilinear

a⁢(u,v)=∫Iu′⁢v′⁢𝑑x (1.108)

e L⁢(v) é a forma linear

L⁢(v)=∫If⁢v⁢𝑑x. (1.109)
Exemplo 1.3.1.

Consideremos o problema

-u′′=1,x∈I=[0,1], (1.110)
u⁢(0)=1/2,u⁢(1)=1. (1.111)

Sua solução analítica é u⁢(x)=-x2/2+x+1/2.

Para obtermos uma aproximação de elementos finitos, consideramos o seguinte problema fraco: encontrar u∈V:={v∈H1(I);v(0)=1/2,v(1)=1} tal que

a⁢(u,v)=L⁢(v), (1.112)

para todo v∈V0={v∈H1(I);v(0)=v(1)=0}, onde

a⁢(u,v) =∫Iu′⁢v′⁢𝑑x,L⁢(v) =∫If⁢v⁢𝑑x. (1.113)

Então, o problema de elementos finitos no espaço das funções lineares por partes lê-se: encontrar uh∈Vh={v∈P1(I);v(0)=1/2,v(1)=1} tal que

a⁢(uh,vh)=L⁢(vh), (1.114)

para todo vh∈Vh,0={v∈H1(I);v(0)=v(1)=0}.

A Figura 1.9 apresenta o esboço dos gráficos da solução analítica u e da sua aproximação de elementos finitos uh, esta construída no espaço dos polinômios lineares por partes sobre uma malha uniforme de 5 células.

Com o FENiCS, a computação do problema de elementos finitos pode ser feita com o seguinte código:

from __future__ import print_function, division
from fenics import *
import numpy as np
import matplotlib.pyplot as plt

#tolerance
tol=1e-14

# malha
mesh = IntervalMesh(5,0,1)

# espaco
V = FunctionSpace(mesh, ’P’, 1)

u_D = Expression(’x[0]<0.5 ? 0.5 : 1’,degree=1)

def boundary(x,on_boundary):
    return on_boundary

bc = DirichletBC(V,u_D,boundary)

#MEF problem
u = TrialFunction(V)
v = TestFunction(V)
f = Constant(1.0)
a = u.dx(0)*v.dx(0)*dx
L = f*v*dx

#computa a sol
u = Function(V)
solve(a == L, u, bc)

#sol analitica
ua = Expression(’-x[0]*x[0]/2+x[0]+0.5’,
                degree=2)

#erro L2
print("Erro L2: %1.2E\n" % errornorm(u,ua,norm_type="L2"))

plot(u,mesh=mesh,marker=’o’,label=r"$u_h$")
mesh = IntervalMesh(100,0,1)
plot(ua,mesh=mesh,label=r"$u$")
plt.legend(numpoints=1)
plt.show()

1.3.2 Condições de Neumann

Consideremos o seguinte problema com condições de contorno de Neumann55Carl Gottfried Neumann, 1832 - 1925, matemático alemão. Fonte: Wikipedia. homogênea em x=L: encontrar u tal que

-u′′=f,∀x∈I=[0,L], (1.115)
u⁢(0)=u0,u′⁢(L)=0, (1.116)

com u0 e f dados.

Tomando uma função teste v∈V:={v∈H1(I);v(0)=0} e multiplicando-a em (1.115), obtemos

-∫Iu′′⁢v⁢𝑑x=∫If⁢v⁢𝑑x. (1.117)

Aplicando a integração por partes, temos

∫Iu′⁢v′⁢𝑑x-u′⁢(L)⁢v⁢(L)⏟u′⁢(L)=0+u′⁢(0)⁢v⁢(0)⏟v⁢(0)=0=∫If⁢c⁢𝑑x. (1.118)

Desta forma, definimos o seguinte problema fraco associado: encontrar u∈V~:={v∈H1(I);v(0)=u0} tal que

a⁢(u,v)=L⁢(v),∀v∈V, (1.119)

onde a⁢(u,v) é a forma bilinear

a⁢(u,v)=∫Iu′⁢v′⁢𝑑x (1.120)

e L⁢(v) é a forma linear

L⁢(v)=∫If⁢v⁢𝑑x. (1.121)
Exemplo 1.3.2.

Consideremos o problema

-u′′=1,x∈I=[0,1], (1.122)
u⁢(0)=0,u′⁢(1)=0. (1.123)

Sua solução analítica é u⁢(x)=-x2/2+x.

Podemos construir uma aproximação por elementos finitos do seguinte problema fraco associado: encontrar u∈V={v∈H1(I);v(0)=0} tal que

a⁢(u,v)=L⁢(v), (1.124)

para todo v∈V, com as formas bilinear a⁢(⋅,⋅) e linear L⁢(⋅) dadas em (1.120) e (1.121).

Então, considerando elementos lineares por partes, temos o seguinte problema de elementos finitos: encontrar uh∈Vh={vh∈P1(I);vh(0)=0} tal que

a⁢(uh,vh)=L⁢(vh),∀vh∈Vh. (1.125)

Agora, consideremos o seguinte problema com condições de Neumann não-homogênea em x=L: encontrar u tal que

-u′′=f,∀x∈I=[0,L], (1.126)
u⁢(0)=u0,u′⁢(L)=α, (1.127)

com u0, α e f dados.

Tomando uma função teste v∈V:={v∈H1(I);v(0)=0} e multiplicando-a em (1.126), obtemos

-∫Iu′′⁢v⁢𝑑x=∫If⁢v⁢𝑑x. (1.128)

Aplicando a integração por partes, temos

∫Iu′⁢v′⁢𝑑x-α⁢v⁢(L)=∫If⁢c⁢𝑑x. (1.129)

Desta forma, definimos o seguinte problema fraco associado: encontrar u∈V~:={v∈H1(I);v(0)=u0} tal que

a(u,v)-b(=L(v),∀v∈V, (1.130)

onde a⁢(u,v) é a forma bilinear

a⁢(u,v)=∫Iu′⁢v′⁢𝑑x (1.131)

e L⁢(v) é a forma linear

L⁢(v)=∫If⁢v⁢𝑑x+α⁢v⁢(L). (1.132)
Exemplo 1.3.3.

Consideremos o problema

-u′′=1,x∈I=[0,1], (1.133)
u⁢(0)=0,u′⁢(1)=1. (1.134)

Sua solução analítica é u⁢(x)=-x2/2+2⁢x.

Agora, consideramos o seguinte problema fraco associado: encontrar u∈V={v∈H1(I);v(0)=0} tal que

a⁢(u,v)=L⁢(v),∀v∈V, (1.135)

com

a⁢(u,v)=∫Iu′⁢v′⁢𝑑x (1.136)

e

L⁢(v)=∫If⁢v⁢𝑑x+1⋅v⁢(1). (1.137)

Então, consideramos o seguinte problema de elementos finitos associado: encontrar uh∈Vh={vh∈P1(I);vh(0)=0} tal que

a⁢(uh,vh)=L⁢(vh),∀vh∈Vh. (1.138)

1.3.3 Condições de Robin

Consideremos o seguinte problema com condições de contorno de Robin66Victor Gustave Robin, 1855 - 1897, matemático francês. Fonte: Wikipedia.: encontrar u tal que

-u′′=f,∀x∈I=[0,L], (1.139)
u′⁢(0)=r0⁢(u⁢(0)-s0),-u′⁢(L)=rL⁢(u⁢(L)-sL), (1.140)

com r0, rL, s0, sL e f dados.

Tomando uma função teste v∈V=H1⁢(I) e multiplicando-a em (1.139), obtemos

-∫Iu′′⁢v⁢𝑑x=∫If⁢v⁢𝑑x. (1.141)

Aplicando a integração por partes, temos

∫Iu′⁢v′⁢𝑑x-u′⁢(L)⁢v⁢(L)⏟-u′⁢(L)=rL⁢(u⁢(L)-sL)+u′⁢(0)⁢v⁢(0)⏟u′⁢(0)=r0⁢(u⁢(0)-s0)=∫If⁢c⁢𝑑x. (1.142)

ou, mais adequadamente,

∫Iu′⁢v′⁢𝑑x+rL⁢u⁢(L)⁢v⁢(L)+r0⁢u⁢(0)⁢v⁢(0)=∫If⁢c⁢𝑑x+rL⁢sL⁢v⁢(L)+r0⁢s0⁢v⁢(0). (1.143)

Desta forma, definimos o seguinte problema fraco associado: encontrar u∈H1⁢(I) tal que

a⁢(u,v)=L⁢(v),∀v∈V, (1.144)

onde a⁢(u,v) é a forma bilinear

a⁢(u,v)=∫Iu′⁢v′⁢𝑑x+rL⁢u⁢(L)⁢v⁢(L)+r0⁢u⁢(0)⁢v⁢(0) (1.145)

e L⁢(v) é a forma linear

L⁢(v)=∫If⁢v⁢𝑑x+rL⁢sL⁢v⁢(L)+r0⁢s0⁢v⁢(0). (1.146)
Exemplo 1.3.4.

Consideremos o problema

-u′′=1,x∈I=[0,1], (1.147)
u′⁢(0)=u⁢(0),-u′⁢(1)=u⁢(1)-1. (1.148)

Sua solução analítica é u⁢(x)=-x2/2+5⁢x/6+5/6.

Aqui, tomamos o seguinte problema fraco: encontrar u∈V=H1⁢(I) tal que

a⁢(u,v)=L⁢(v),∀v∈V, (1.149)

onde

a⁢(u,v)=∫Iu′⁢v′⁢𝑑x+u⁢(1)⁢v⁢(1)+u⁢(0)⁢v⁢(0) (1.150)

e

L⁢(v)=∫If⁢v⁢𝑑x+1⋅v⁢(1). (1.151)

Então, uma aproximação por elementos finitos lineares por partes pode ser obtida resolvendo o seguinte problema: encontrar uh∈Vh=P1⁢(I) tal que

a⁢(uh,vh)=L⁢(vh),∀vh∈Vh. (1.152)

A Figura 1.12 apresenta o esboço dos gráficos da solução analítica u e da sua aproximação de elementos finitos uh, esta construída no espaço dos polinômios lineares por partes sobre uma malha uniforme de 5 células.

Com o FENiCS, a computação do problema de elementos finitos pode ser feita com o seguinte código:

from __future__ import print_function, division
from fenics import *
import numpy as np
import matplotlib.pyplot as plt

#tolerance
tol=1e-14

# malha
mesh = IntervalMesh(5,0,1)

# espaco
V = FunctionSpace(mesh, ’P’, 1)

#marcadores de fronteiras
boundary_markers = FacetFunction(’size_t’, mesh)

class BoundaryX0(SubDomain):
    def inside(self, x, on_boundary):
        return on_boundary and near(x[0], 0, tol)
bx0 = BoundaryX0()
bx0.mark(boundary_markers, 0)

class BoundaryX1(SubDomain):
    def inside(self, x, on_boundary):
        return on_boundary and near(x[0], 1, tol)
bx1 = BoundaryX1()
bx1.mark(boundary_markers, 1)

ds = Measure(’ds’, domain=mesh, \
                 subdomain_data=boundary_markers)

#MEF problem
u = TrialFunction(V)
v = TestFunction(V)
f = Constant(1.0)
a = u.dx(0)*v.dx(0)*dx + u*v*ds(1) + u*v*ds(0)
L = f*v*dx + 1*v*ds(1)

#computa a sol
u = Function(V)
solve(a == L, u)

#sol analitica
ua = Expression(’-x[0]*x[0]/2+5*x[0]/6+5./6’,
                degree=2)

#erro L2
print("Erro L2: %1.2E\n" % errornorm(u,ua,norm_type="L2"))

plot(u,mesh=mesh,marker=’o’,label=r"$u_h$")
mesh = IntervalMesh(100,0,1)
plot(ua,mesh=mesh,label=r"$u$")
plt.legend(numpoints=1,loc=’upper left’)
plt.show()
