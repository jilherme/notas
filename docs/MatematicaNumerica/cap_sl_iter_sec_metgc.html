4.3 Método do gradiente conjugado

O método do gradiente conjugado é uma variação do método do gradiente (veja Seção 4.2). Aqui, a solução de um dado sistema A⁢𝒙=𝒃, com A uma matriz positiva definida, é computada de forma iterativa por

𝒙(1) =aprox. inicial, (4.91)
𝒅(1) =𝒓(1), (4.92)
𝒙(k+1) =𝒙(k)+αk⁢𝒅(k), (4.94)
α(k) =-𝒓(k)⋅𝒅(k)𝒅(k)⋅A⁢𝒅(k), (4.95)
𝒅(k+1) =-𝒓(k+1)+βk⁢𝒅(k), (4.96)
β(k) =𝒓(k+1)⋅A⁢𝒅(k)𝒅(k)⋅A⁢𝒅(k), (4.97)

para k=1,2,…, e 𝒓(k)=A⁢𝒙(k)-𝒃.

Exemplo 4.3.1.

Consideremos o sistema A⁢x=b com

A=[2-100-12-100-12-100-12],b=[-322-3]. (4.98)

Na Tabela 4.5 temos os resultados do emprego do método do gradiente conjugado com 𝒙(1)=(0,0,0,0).

No GNU Octave, podemos fazer as computações acima com o seguinte código:

A = [2 -1 0 0;...
     -1 2 -1 0;...
     0 -1 2 -1;...
     0 0 -1 2]
x = [-1 1 1 -1]’;
b = [-3 2 2 -3]’

x = [0 0 0 0]’
r = A*x - b;
d = r;
disp([x’,norm(r)])

for k=2:3
  alpha = r’*d/(d’*A*d);
  x = x - alpha*d;
  r = A*x - b;
  disp([x’,norm(r)])
  beta=r’*A*d/(d’*A*d);
  d=-r+beta*d;
endfor
