4.2 Método do gradiente

Começamos observando que se A é uma matriz n×n positiva definida2020A é simétrica e xT⁢A⁢x>0 para todo x≠0., temos que 𝒙∈ℝn é solução de

A⁢𝒙=𝒃 (4.40)

se, e somente se, 𝒙 é solução do seguinte problema de minimização

min𝒙∈ℝn⁡f⁢(𝒙):=12⁢𝒙T⁢A⁢𝒙-𝒙T⁢𝒃. (4.41)

O método do gradiente é um algoritmo da forma: dada uma aproximação inicial 𝒙(1) da solução de (4.41) (ou, equivalentemente, de (4.40)), computamos novas aproximações da forma iterativa

𝒙(k+1)=𝒙(k)+α(k)⁢𝒅(k),k=1,2,⁢…, (4.42)

onde α(k) é o tamanho do passo (um escalar) e 𝒅(k)∈ℝn é a direção de busca.

Para escolhermos a direção 𝒅(k), tomamos a fórmula de Taylor de f em torno da aproximação 𝒙(k)

f⁢(𝒙(k+1))=f⁢(𝒙(k))+α(k)⁢∇⁡f⁢(𝒙(k))⋅𝒅(k)+O⁢((α(k))2), (4.43)

com α(k)→0, onde ∇⁡f denota o gradiente de f, i.e.

∇⁡f⁢(𝒙(k)) =(∂⁡f∂⁡x1⁢(𝒙(k)),∂⁡f∂⁡x2⁢(𝒙(k)),…,∂⁡f∂⁡xn⁢(𝒙(k))) (4.44)
=A⁢𝒙(k)-𝒃. (4.45)

De (4.43), segue que se

∇⁡f⁢(𝒙(k))⋅𝒅(k)<0, (4.46)

então f⁢(𝒙(k+1))<f⁢(𝒙(k)) se α(k) é suficientemente pequeno. Em particular, podemos escolher

𝒅(k)=-∇⁡f⁢(𝒙(k)), (4.47)

se ∇⁡f⁢(𝒙(k))≠0.

Do exposto acima, temos a iteração do método do gradiente

𝒙(1) =aprox. inicial (4.48)
𝒙(k+1) =𝒙(k)-α(k)⁢𝒓(k),k=1,2,⁢…, (4.49)

onde 𝒓(k) é o resíduo da iterada k dado por

𝒓(k)=A⁢𝒙(𝒌)-𝒃. (4.50)
Exemplo 4.2.1.

Consideremos o sistema A⁢x=b com

A=[2-100-12-100-12-100-12],b=[-322-3]. (4.51)

Na Tabela 4.3 temos os resultados do emprego do método do gradiente com 𝒙(1)=(0,0,0,0) e com passo constante α(k)≡0,5.

No GNU Octave, podemos fazer as computações acima com o seguinte código:

A = [2 -1 0 0;...
     -1 2 -1 0;...
     0 -1 2 -1;...
     0 0 -1 2]
b = [-3 2 2 -3]’

alpha=5e-1;
x = [0 0 0 0]’
r = A*x - b;
disp([x’,norm(r)])

for k=2:11
  x = x - alpha*r;
  r = A*x - b;
  disp([x’,norm(r)])
endfor

4.2.1 Escolha do passo

Da iteração do método do gradiente, temos que a melhor escolha do passo α(k) é tal que

f⁢(𝒙(k)+α(k)⁢𝒓(k))=minα>0⁡f⁢(𝒙(k)+α⁢𝒓(k)). (4.52)

Desta forma,

dd⁢α⁢f⁢(𝒙(k)+α⁢𝒓(k))=0 ⇒∇⁡f⁢(𝒙(k+1))⋅𝒓(k)=0, (4.53)
⇒(A⁢(𝒙(k)+α(k)⁢𝒓(k))-b)⋅𝒓(k)=0, (4.54)
⇒(A⁢𝒙(k)-𝒃)⋅𝒓(k)+α(k)⁢𝒓(k)⋅A⁢𝒓(k)=0, (4.55)

donde

α(k)=-𝒓(k)⋅𝒓(k)𝒓(k)⋅A⁢𝒓(k). (4.56)
Exemplo 4.2.2.

Consideremos o sistema A⁢x=b com

A=[2-100-12-100-12-100-12],b=[-322-3]. (4.57)

Na Tabela 4.4 temos os resultados do emprego do método do gradiente com 𝒙(1)=(0,0,0,0) e com passo

α(k)=-𝒓(k)⋅𝒓(k)𝒓(k)⋅A⁢𝒓(k). (4.58)

No GNU Octave, podemos fazer as computações acima com o seguinte código:

A = [2 -1 0 0;...
     -1 2 -1 0;...
     0 -1 2 -1;...
     0 0 -1 2]
x = [-1 1 1 -1]’;
b = [-3 2 2 -3]’

x = [0 0 0 0]’
r = A*x - b;
disp([x’,norm(r)])

for k=2:5
  alpha = r’*r/(r’*A*r);
  x = x - alpha*r;
  r = A*x - b;
  disp([x’,norm(r)])
endfor
