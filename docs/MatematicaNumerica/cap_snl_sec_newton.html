5.1 Método de Newton

Consideremos o problema de encontrar 𝒙=(x1,x2,…,xn)∈ℝn tal que

F⁢(𝒙)=0, (5.2)

onde F:ℝn→ℝn é uma função vetorial dada com F⁢(𝒙)=(f1⁢(𝒙),f2⁢(𝒙),…,fn⁢(𝒙)). Suponhamos, então, que 𝒙* seja a solução exata deste problema e que 𝒙(1) seja uma aproximação de 𝒙*. Assim sendo, tomemos a seguinte expansão de F em polinômio de Taylor:

F⁢(𝒙*)=F⁢(𝒙(1))+JF⁢(𝒙(1))⁢(𝒙*-𝒙(1))+R, (5.3)

onde JF é a matriz jacobiana2121Carl Gustav Jacob Jacobi, 1804 - 1851, matemático alemão. Fonte: Wikipedia. de F

JF:=[∂⁡f1∂⁡x1∂⁡f1∂⁡x2…∂⁡f1∂⁡xn∂⁡f2∂⁡x1∂⁡f2∂⁡x2…∂⁡f2∂⁡xn⋮⋮⋮⋮∂⁡fn∂⁡x1∂⁡fn∂⁡x2…∂⁡fn∂⁡xn] (5.4)

e ∥R∥2→0 com ∥𝒙(1)-𝒙*∥→0.

Daí, como F⁢(𝒙*)=𝟎, temos

JF⁢(𝒙(1))⁢(𝒙*-𝒙(1))≈-F⁢(𝒙(1)). (5.5)

Então, multiplicando a inversa da jacobiana à esquerda, obtemos

𝒙*-𝒙(1)≈-JF-1⁢(𝒙(1))⁢F⁢(𝒙(1)) (5.6)

e também

𝒙*≈𝒙(1)-JF-1⁢(𝒙(1))⁢F⁢(𝒙(1)). (5.7)

O exposto acima nos motiva a iteração de Newton2222Sir Isaac Newton, 1642 - 1726/27, matemático e físico inglês. Fonte: Wikipedia.:

𝒙(1) =aprox. inicial, (5.8)
𝒙(k+1) =𝒙(k)-JF-1⁢(𝒙(k))⁢F⁢(𝒙(k)), (5.9)

com k=1,2,⁢….

Exemplo 5.1.1.

Consideremos o seguinte sistema de equações não lineares

x1⁢x22 =x12⁢x2-6, (5.10)
x12⁢x23-7 =-x1. (5.11)

Para usarmos o método de Newton, reescrevemos o sistema na seguinte forma

x1⁢x22-x12⁢x2+6 =0, (5.12)
x1+x12⁢x23-7 =0. (5.13)

Com isso, identificamos a função objetivo

F⁢(𝒙)=[f1⁢(𝒙)f2⁢(𝒙)]=[x1⁢x22-x12⁢x2+6x1+x12⁢x23-7] (5.14)

e sua matriz jacobiana

JF⁢(𝒙) =∂⁡(f1,f2)∂⁡(x1,x2)=[∂⁡f1∂⁡x1∂⁡f1∂⁡x2∂⁡f2∂⁡x1∂⁡f2∂⁡x2] (5.15)
=[x22-2⁢x1⁢x22⁢x1⁢x2-x121+2⁢x1⁢x233⁢x12⁢x22] (5.16)

Definidas F e JF e tomando 𝒙(1)=(1,5, 1,5) como aproximação inicial, computamos as iterações de Newton de forma a obtermos os resultados apresentados na Tabela 5.1.

No GNU Octave, podemos fazer as computações acima com o seguinte código:

disp(’Newton’)

function y = F(x)
  y = zeros(size(x));
  y(1) = x(1)*x(2)^2 - x(1)^2*x(2)+6;
  y(2) = x(1) + x(1)^2*x(2)^3-7;
endfunction

function y = J(x)
  y=zeros(2,2);
  y(1,1) = x(2)^2-2*x(1)*x(2);
  y(1,2) = 2*x(1)*x(2)-x(1)^2;
  y(2,1) = 1+2*x(1)*x(2)^3;
  y(2,2) = 3*x(1)^2*x(2)^2;
endfunction

x=[-1.5 1.5]’;
printf("%d %1.2E %1.2E %1.1E\n",1,x,norm(F(x)))
for k=2:5
  x = x - inv(J(x))*F(x);
  printf("%d %1.2E %1.2E %1.1E\n",k,x,norm(F(x)))
endfor

5.1.1 Considerações sobre convergência

Para uma função F suficientemente suave e com uma escolha apropriada da aproximação inicial 𝒙(1), temos que as iterações de Newton

𝒙(k+1)=𝒙(k)-JF-1⁢(𝒙(1))⁢F⁢(𝒙(1)), (5.17)

são quadraticamente convergentes, i.e.

∥𝒙(k+1)-𝒙*∥≤C⁢∥𝒙(k)-𝒙*∥2, (5.18)

onde 𝒙* é a solução exata, i.e. F⁢(𝒙*)=𝟎.

Exemplo 5.1.2.

Consideremos o seguinte sistema de equações não lineares

x1⁢x22-x12⁢x2+6 =0, (5.19)
x1+x12⁢x23-7 =0. (5.20)

A Figura 5.1 é um esboço do gráfico da ∥F⁢(⋅)∥. Este problema foi confeccionado de forma que 𝒙*=(-1, 2). Então, tomando 𝒙(1)=(1,5, 1,5) como aproximação inicial, computamos as iterações de Newton para este problema, donde obtemos os resultados reportados na Tabela 5.2.

No GNU Octave, podemos fazer as computações acima com o seguinte código:

disp(’Newton’)

function y = F(x)
  y = zeros(size(x));
  y(1) = x(1)*x(2)^2 - x(1)^2*x(2)+6;
  y(2) = x(1) + x(1)^2*x(2)^3-7;
endfunction

function y = J(x)
  y=zeros(2,2);
  y(1,1) = x(2)^2-2*x(1)*x(2);
  y(1,2) = 2*x(1)*x(2)-x(1)^2;
  y(2,1) = 1+2*x(1)*x(2)^3;
  y(2,2) = 3*x(1)^2*x(2)^2;
endfunction

xx1 = linspace(-2,2.25);
xx2 = xx1;
zz = zeros(length(xx1),length(xx2));
for i = 1:length(xx1)
  for j = 1:length(xx2)
    zz(j,i) = norm(F([xx1(i),xx2(j)]));
  endfor
endfor

meshc(xx1,xx2,zz)
xlabel("x1")
ylabel("x2")

xstar = [-1 2]’;
x=[-1.5 1.5]’;
printf("%d %1.2E %1.2E %1.1E\n",1,x,norm(x-xstar))
for k=2:5
  x = x - inv(J(x))*F(x);
  printf("%d %1.2E %1.2E %1.1E\n",k,x,norm(x-xstar))
endfor
