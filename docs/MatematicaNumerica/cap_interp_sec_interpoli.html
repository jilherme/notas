6.1 Interpolação polinomial

Dado um conjunto de n pontos {(xi,yi)}i=1n, o problema de interpolação consiste em encontrar o polinômio de grau n-1

p⁢(x)=p1⁢xn-1+p2⁢xn-2+⋯+pn-1⁢x+pn (6.3)

tal que

yi=p⁢(xi),i=1,2,…,n. (6.4)

Das condições (6.3), temos

p1⁢x1n-1+p2⁢x1n-2+⋯+pn-1⁢x1+pn =y1 (6.5)
p1⁢x2n-1+p2⁢x2n-2+⋯+pn-1⁢x2+pn =y2 (6.6)
⋮ (6.7)
p1⁢xnn-1+p2⁢xnn-2+⋯+pn-1⁢xn+pn =yn. (6.8)

Isto é, os coeficientes do polinômio interpolador (6.3) satisfazem o seguinte sistema linear

A⁢𝒑=𝒚, (6.9)

onde A é a matriz de Vandermonde11 1 Alexandre-Théophile Vandermonde, 1735 - 1796, matemático francês. Fonte: Wikipedia.

A=[x1n-1x1n-2…x11x2n-1x2n-2…x21⋮⋮⋮⋮⋮xnn-1xnn-2…xn1], (6.10)

𝒑=(p1,p2,…,pn) é o vetor das incógnitas e 𝒚=(y1,y2,…,yn) é o vetor dos termos constantes.

Exemplo 6.1.1.

Consideremos o problema de encontrar o polinômio interpolador do conjunto de pontos {(-1,-1),(0, 1),(1, 1/2)}. Como temos 3 pontos, o polinômio tem grau 2 e pode ser escrito na forma

p⁢(x)=p1⁢x2+p2⁢x+p3. (6.11)

Seguindo a abordagem acima, temos 𝒑=(p1,p2,p3), 𝒙=(-1,0,1), 𝒚=(-1,1,1/2) e

A=[x12x11x22x21x32x31]. (6.12)

Então, resolvendo A⁢𝒑=𝒚, obtemos o polinômio interpolador

p⁢(x)=-1,25⁢x2+0,75⁢x+1. (6.13)

A Figura 6.1 mostra os esboços do polinômio interpolador p⁢(x) e dos pontos dados.

No GNU Octave, podemos fazer as computações acima com o seguinte código:

x = [-1 0 1]’;
y = [-1 1 0.5]’;
A = [x.^2 x.^1 x.^0]
p = inv(A)*y

xx=linspace(-1.25,1.25);
plot(x,y,’ro’,...
     xx,polyval(p,xx),’b-’)
