3.4 EDO de ordem mais alta

Os métodos aplicados para EDOs lineares de segunda ordem podem ser estendidos para tratarmos EDOs lineares de ordem mais alta. Aqui, vamos nos restringir ao caso de tais EDOs com coeficientes constantes, i.e. equações da forma

an⁢y(n)+an-1⁢y(n-1)+⋯+a0⁢y=g⁢(x), (3.290)

onde y=y⁢(t), a0, a1, …, an são constantes dadas e an≠0.

3.4.1 EDO homogênea

A solução geral de uma EDO homogênea da forma

an⁢y(n)+an-1⁢y(n-1)+⋯+a0⁢y=0, (3.291)

é dada por

y⁢(t)=c1⁢y1⁢(t)+c2⁢y2⁢(t)+⋯+cn⁢yn⁢(t), (3.292)

sendo que y1, y2, …, yn formam um conjunto fundamental de soluções, i.e. são soluções tais que o wronskiano99 9 Józef Maria Hoene-Wroński, 1776 - 1853, matemático polonês. Fonte: Wikipedia.

W⁢(y1,y2,…,yn;t)=|y1y2⋯yny1′y2′⋯yn′⋮⋮⋮⋮y1(n-1)y2(n-1)⋯yn(n-1)|≠0. (3.293)

Podemos obter soluções particulares da forma

y⁢(t)=er⁢t. (3.294)

Substituindo na EDO, vemos que r deve satisfazer a equação característica

an⁢rn+an-1⁢rn-1+a0=0. (3.295)

Cada raiz nos fornece uma solução particular yp=yp⁢(t):

  1. a)

    se r=rp é raiz simples, então yp⁢(t)=erp⁢t

  2. b)

    se r=rp raiz de multiplicidade m, então

    yp+1⁢(t) =erp⁢t (3.296)
    yp+2⁢(t) =t⁢erp⁢t (3.297)
    ⋮ (3.298)
    yp+m =tm-1⁢erp⁢t (3.299)
  3. c)

    se r=λp±i⁢μp é raiz complexa, então

    yp+1 =eλp⁢t⁢cos⁡(μp⁢t) (3.300)
    yp+2 =eλp⁢t⁢sen⁡(μp⁢t). (3.301)
Exemplo 3.4.1.

Vamos calcular a solução geral de

y′′′+2⁢y′′-y′-2⁢y=0. (3.302)

A equação característica associada é

r3+2⁢r2-r-2=0. (3.303)

Queremos calcular as raízes do polinômio característico. Para tanto, começamos observando que r=1 é raiz, pois 13+2⋅12-1-2=0. Logo, o polinômio é divisível pelo monômio (x-1). Calculando a divisão

 ⁢r3+2⁢r2-r-2|r-1 ¯ (3.304)
-r3+r2¯     r2+3⁢r+2 (3.305)
   3⁢r2-r (3.306)
 ⁢-3⁢r2+3⁢r¯ (3.307)
    2⁢r-2 (3.308)
   -2⁢r+2¯ (3.309)
      0 (3.310)

obtemos, para r≠1,

r3+2⁢r2-r-2r-1=r2+3⁢r+2 (3.311)

ou, equivalentemente,

r3+2⁢r2-r-2=(r-1)⁢(r2+3⁢r+2) (3.312)

Portanto, as outras raízes do polinômio característico ocorrem quando

r2+3⁢r+2=0 (3.313)
r=-3±9-4⋅1⋅212 (3.314)
r=-2,ou r=-1. (3.315)

Concluímos que as soluções da equação característica são r1=-2, r2=-1 e r3=1. Logo, a solução geral é

y⁢(t)=c1⁢e-2⁢t+c2⁢e-t+c3⁢et. (3.316)

Vamos, ainda, verificar se y1⁢(t)=e-2⁢t, y2⁢(t)=e-t e y3⁢(t)=et formam um conjunto fundamental de soluções. Por construção, sabemos que estas são soluções particulares. Resta, portanto, verificar que o wronskiano é não nulo. De fato, temos

W⁢(y1,y2,y3;t) =|y1y2y3y1′y2′y3′y1′′y2′′y3′′| (3.317)
=|e-2⁢te-tet-2⁢e-2⁢t-e-tet4⁢e-2⁢te-tet| (3.318)
=6⁢e-2⁢t≠0,∀t. (3.319)

3.4.2 Equação não homogênea

Ambos os métodos da variação dos parâmetros e dos coeficientes a determinar podem ser generalizados para EDOs lineares de ordem mais altas, com coeficientes constantes e não homogêneas. Tais equações têm a forma

y(n)+an-1⁢y(n-1)+⋯+a0⁢y=g⁢(t), (3.327)

com n≥1 e g⁢(t)≢0. A solução geral pode ser escrita na forma

y⁢(t)=c1⁢y1⁢(t)+c2⁢y2⁢(t)+⋯+cn⁢yn⁢(t)+yp⁢(t), (3.328)

onde y1, y2, …, yn formam um conjunto fundamental de soluções para a equação homogênea associada e yp é uma solução particular qualquer da equação não homogênea.

Método da variação dos parâmetros

Seja a equação diferencial ordinária

y(n)+an-1⁢y(n-1)+⋯+a0⁢y=g⁢(t), (3.329)

com n≥1 e g⁢(t)≢0. Seja, também, y1, y2, …, yn um conjunto fundamental de soluções da equação homogênea associada. O método da variação dos parâmetros consiste em buscar uma solução particular para (3.329) com o seguinte formato

yp⁢(t)=u1⁢(t)⁢y1⁢(t)+u2⁢(t)⁢y2⁢(t)+⋯+un⁢(t)⁢yn⁢(t), (3.330)

onde u1, u2, …, un são funções parâmetros a determinar.

Os parâmetros u1, u2, …, un devem ser escolhidos de forma que yp satisfaça (3.329). Quando n>1, temos um problema subdeterminado (mais variáveis que equações). Para fechar o problema, exigimos que os parâmetros sejam tais que

u1′⁢y1+u2′⁢y2+⋯+un′⁢yn =0 (3.331)
u1′⁢y1′+u2′⁢y2′+⋯+un′⁢yn′ =0 (3.332)
⋮ (3.333)
u1′⁢y1(n-2)+u2′⁢y2(n-2)+⋯+un′⁢yn(n-2) =0. (3.334)

Com isso, ao substituirmos yp em (3.329), obtemos

g⁢(t) =a0⁢yp+a1⁢yp′+a2⁢yp′′+⋯+yp(n) (3.335)
=a0⁢(u1⁢y1+u2⁢y2+⋯+un⁢yn) (3.336)
+a1⁢(u1⁢y1′+u2⁢y2′+⋯+un⁢yn′) (3.337)
+a2⁢(u1⁢y1′′+u2⁢y2′′+⋯+un⁢yn′′) (3.338)
+⋯+ (3.339)
+(u1⁢y1(n)+u2⁢y2(n)+⋯+un⁢yn(n)) (3.340)
+(u1′⁢y1(n-1)+u2′⁢y2(n-1)+⋯+un′⁢yn(n-1)). (3.341)

Lembrando que y1, y2, …, yn são soluções da equação homogênea associada, esta última equação é equivalente a

u1′⁢y1(n-1)+u2′⁢y2(n-1)+⋯+un′⁢yn(n-1)=g⁢(t). (3.342)

Desta forma, concluímos que os parâmetros podem ser escolhidos de forma a satisfazerem o sistema de equações (3.331)-(3.334) e (3.342), i.e.

u1′⁢y1+u2′⁢y2+⋯+un′⁢yn =0 (3.343)
u1′⁢y1′+u2′⁢y2′+⋯+un′⁢yn′ =0 (3.344)
⋮ (3.345)
u1′⁢y1(n-2)+u2′⁢y2(n-2)+⋯+un′⁢yn(n-2) =0 (3.346)
u1′⁢y1(n-1)+u2′⁢y2(n-1)+⋯+un′⁢yn(n-1) =g. (3.347)

Usando o método de Cramer1212 12 Gabriel Cramer, 1704 - 1752, matemático suíço. Fonte: Wikipedia. temos

um′⁢(t)=g⁢(t)⁢Wm⁢(t)W⁢(t), (3.348)

onde m=1,2,…,n, W⁢(t) denota o wronskiano

W⁢(y1,y2,⋯,yn;t)=|y1y2⋯yny1′y2′⋯yn′⋮⋮⋯⋮y1(n-1)y2(n-1)⋯yn(n-1)| (3.349)

e Wm⁢(t) é o determinante obtido de W substituindo a m-ésima coluna por vetor (0,0,…⁢,1).

Por fim, integrando (3.348), obtemos os parâmetros

um⁢(t)=∫g⁢(t)⁢Wm⁢(t)W⁢(t)⁢𝑑t, (3.350)

com m=1,2,…,n.

Exemplo 3.4.3.

Vamos calcular a solução geral de

y′′′+2⁢y′′-y′-2⁢y=e2⁢t. (3.351)

No Exemplo 3.4.1, vimos que

y1⁢(t)=e-2⁢t,y2⁢(t)=e-t,y3⁢(t)=et (3.352)

formam um conjunto fundamental de soluções para a equação homogênea associada. Com isso, nos resta calcular uma solução particular yp=yp⁢(t) para a equação não homogênea.

Pelo método da variação dos parâmetros, podemos calcular como

yp⁢(t)=u1⁢(t)⁢y1⁢(t)+u2⁢(t)⁢y2⁢(t)+u3⁢(t)⁢y3⁢(t), (3.353)

onde

um⁢(t)=∫g⁢(t)⁢Wm⁢(t)W⁢(t)⁢𝑑t, (3.354)

sendo g⁢(t)=e2⁢t e m=1,2,3. Ainda, temos

W⁢(t) =|y1y2y3y1′y2′y3′y1′′y2′′y3′′| (3.355)
=|e-2⁢te-tet-2⁢e-2⁢t-e-tet4⁢e-2⁢te-tet| (3.356)
=6⁢e-2⁢t (3.357)
W1⁢(t) =|0y2y30y2′y3′1y2′′y3′′| (3.358)
=(-1)3+1⁢|y2y3y2′y3′| (3.359)
=|e-tet-e-tet| (3.360)
=2 (3.361)
W2⁢(t) =|y10y3y1′0y3′y1′′1y3′′| (3.362)
=(-1)3+2⁢|y1y3y1′y3′| (3.363)
=-|e-2⁢tet-2⁢e-2⁢tet| (3.364)
=-3⁢e-t (3.365)
W3⁢(t) =|y1y20y1′y2′0y1′′y2′′1| (3.366)
=(-1)3+3⁢|y1y2y1′y2′| (3.367)
=|e-2⁢te-t-2⁢e-2⁢t-e-t| (3.368)

Logo,

u1⁢(t) =∫g⁢(t)⁢W1⁢(t)W⁢(t)⁢𝑑t (3.370)

=112⁢e4⁢t (3.373)
u2⁢(t) =∫g⁢(t)⁢W2⁢(t)W⁢(t)⁢𝑑t (3.374)

=-12⁢∫e3⁢t⁢𝑑t (3.376)
=-16⁢e3⁢t (3.377)
u3⁢(t) =∫g⁢(t)⁢W3⁢(t)W⁢(t)⁢𝑑t (3.378)
=∫e2⁢t⁢e-3⁢t6⁢e-2⁢t⁢𝑑t (3.379)
=16⁢∫et⁢𝑑t (3.380)
=16⁢et (3.381)

Com isso,

yp =u1⁢y1+u2⁢y2+u3⁢y3 (3.382)
=112⁢e4⁢t⁢e-2⁢t-16⁢e3⁢t⁢e-t+16⁢et⁢et (3.383)

Concluímos que a solução geral de (3.351) é

y⁢(t)=c1⁢e-2⁢t+c2⁢e-t+c3⁢et+112⁢e2⁢t. (3.385)

Método dos coeficientes a determinar

A aplicação na resolução de EDOs de ordem mais alta do método dos coeficientes a determinar é análoga ao caso de EDOs de segunda ordem (veja a Subseção 3.3.2).

Exemplo 3.4.4.

Vamos calcular a solução geral de

y′′′+2⁢y′′-y′-2⁢y=e2⁢t. (3.386)

No Exemplo 3.4.1, vimos que

y1⁢(t)=e-2⁢t,y2⁢(t)=e-t,y3⁢(t)=et (3.387)

formam um conjunto fundamental de soluções para a equação homogênea associada. Com isso, nos resta calcular uma solução particular yp=yp⁢(t) para a equação não homogênea.

Com base no termo fonte g⁢(t)=e2⁢t, buscamos por uma solução particular da forma

yp⁢(t)=A⁢e2⁢t, (3.388)

onde A é um coeficiente a determinar.

Para determinar A, substituímos yp na (3.386), donde

e2⁢t =yp′′′+2⁢yp′′-yp′-2⁢yp (3.389)
=8⁢A⁢e2⁢t+8⁢A⁢e2⁢t-2⁢A⁢e2⁢t-2⁢A⁢e2⁢t (3.390)
=12⁢A⁢e2⁢t. (3.391)

Com isso, temos A=1/12 e

yp⁢(t)=112⁢e2⁢t. (3.392)

Concluímos que a solução geral é

y⁢(t)=c1⁢e-2⁢t+c2⁢e-t+c3⁢et+112⁢e2⁢t. (3.393)

ER 3.4.1.

Resolva o PVI

y′′′+3⁢y′′+3⁢y′+y=0,t>0, (3.394)
y⁢(0)=1,y′⁢(0)=0,y′′⁢(0)=0. (3.395)
Solução.

A equação característica associada

r3+3⁢r2+3⁢r+1=0 (3.396)

tem raiz tripla r=-1. Logo, a solução geral é

y⁢(t)=(c1+c2⁢t+c3⁢t2)⁢e-t. (3.397)

Vamos agora aplicar as condições iniciais. Começamos com y⁢(0)=1. Substituindo t=0 na solução geral, obtemos

y⁢(0)=c1, (3.398)

logo, c1=1. Para usarmos a condição inicial y′⁢(0)=0, usamos a derivada da solução geral, i.e.

y′⁢(t)=(-c1+c2+(2⁢c3-c2)⁢t-c3⁢t2)⁢e-t. (3.399)

Então, temos

0=y′⁢(0) (3.400)
0=-c1+c2 (3.401)
0=-1+c2 (3.402)
c2=1. (3.403)

Por fim, para usarmos a condição inicial y′′⁢(0)=0, usamos a segunda derivada da solução geral como segue.

y′′⁢(t)=(2⁢c3-2⁢c2+c1+(c2-4⁢c3)⁢t+c3⁢t2)⁢e-t (3.404)
0=y′′⁢(0)=2⁢c3-2⁢c2+c1 (3.405)
0=2⁢c3-2+1 (3.406)
2⁢c3=1 (3.407)
c3=12. (3.408)

Logo, a solução do PVI é

y⁢(t)=(1+t+12⁢t2)⁢e-t. (3.409)
ER 3.4.2.

Use o método da variação dos parâmetros para calcular a solução geral de

y′′′+3⁢y′′+3⁢y′+y=2⁢t (3.410)
Solução.

No ER 3.4.1, vimos que

y1⁢(t)=e-t, (3.411)
y2⁢(t)=t⁢e-t, (3.412)
y3⁢(t)=t2⁢e-t (3.413)

formam um conjunto fundamental de soluções para a equação homogênea associada a (3.410).

A aplicação do método da variação dos parâmetros consiste em calcular uma solução particular para (3.410) da forma

yp⁢(t)=u1⁢(t)⁢y1⁢(t)+u2⁢(t)⁢y2⁢(t)+u3⁢(t)⁢y3⁢(t). (3.414)

Para calcularmos os parâmetros u1, u2 e u3, precisamos dos seguintes determinantes:

W⁢(y1,y2,y3;t) =|y1y2y3y1′y2′y3′y1′′y2′′y3′′| (3.415)
=2⁢e-3⁢t (3.416)

=t2⁢e-2⁢t (3.418)
W2⁢(t) =|y10y3y1′0y3′y1′′1y3′′| (3.419)
=-2⁢t⁢e-2⁢t (3.420)
W3⁢(t) =|y1y20y1′y2′0y1′′y2′′1| (3.421)
=e-2⁢t (3.422)

Com isso, temos

u1⁢(t) =∫g⁢(t)⁢W1⁢(t)W⁢(t)⁢𝑑t (3.423)

u3⁢(t) =∫g⁢(t)⁢W3⁢(t)W⁢(t)⁢𝑑t (3.429)

yp⁢(t)=2⁢t-6. (3.432)

Concluímos que

y⁢(t)=(c1+c2⁢t+c3⁢t2)⁢e-t+2⁢t-6 (3.433)

No Python1414 14 Veja a Observação 1.0.1., podemos resolver este exercício com o seguinte código:

from sympy import *

# def. das variáveis
t = symbols(’t’)
y = symbols(’y’, cls=Function)

# soluções fundamentais
y1 = exp(-t)
y2 = t*exp(-t)
y3 = t**2*exp(-t)

# wronskiano
WM = Matrix([[y1,y2,y3], \
           [diff(y1,t),diff(y2,t),diff(y3,t)], \
           [diff(y1,t,2),diff(y2,t,2),diff(y3,t,2)]])
W = simplify(WM.det())
print(’W = ’, W)

WM1 = Matrix([[0,y2,y3], \
           [0,diff(y2,t),diff(y3,t)], \
           [1,diff(y2,t,2),diff(y3,t,2)]])
W1 = simplify(WM1.det())
print(’W1 = ’, W1)

WM2 = Matrix([[y1,0,y3], \
           [diff(y1,t),0,diff(y3,t)], \
           [diff(y1,t,2),1,diff(y3,t,2)]])
W2 = simplify(WM2.det())
print(’W2 = ’, W2)

WM3 = Matrix([[y1,y2,0], \
           [diff(y1,t),diff(y2,t),0], \
           [diff(y1,t,2),diff(y2,t,2),1]])
W3 = simplify(WM3.det())
print(’W3 = ’, W3)

# fonte
g = 2*t

# parâmetros
u1 = integrate(g*W1/W,t)
print(’u1(t) = ’, u1)

u2 = integrate(g*W2/W,t)
print(’u2(t) = ’, u2)

u3 = integrate(g*W3/W,t)
print(’u3(t) = ’, u3)

# solucao particular
yp = simplify(u1*y1 + u2*y2 + u3*y3)
print(’yp(t) = ’, yp)

# constantes indeterminadas
C1,C2,C3 = symbols(’C1,C2,C3’)

# sol. geral
y = simplify(C1*y1 + C2*y2 + C3*y3 + yp)
print(’y(t) = ’, y)
ER 3.4.3.

Use o método dos coeficientes a determinar para obter uma solução particular de

y′′′+3⁢y′′+3⁢y′+y=cos⁡(t) (3.434)
Solução.

Tendo em vista o fonte g⁢(t)=cos⁡(t), as soluções fundamentais obtidas no ER 3.4.1 e a Observação 3.3.2, a aplicação do método dos coeficientes a determinar consiste em calcular uma solução particular da forma

yp⁢(t)=A⁢cos⁡(t)+B⁢sen⁡(t), (3.435)

onde A e B são coeficientes a determinar.

Substituindo yp em (3.434), obtemos

cos⁡(t) =yp′′′+3⁢yp′′+3⁢yp′+yp (3.436)
=(2⁢B-2⁢A)⁢cos⁡(t)-(2⁢A+2⁢B)⁢sen⁡(t). (3.437)

Segue que

2⁢B-2⁢A =1 (3.438)
2⁢B+2⁢A =0. (3.439)

Resolvendo, obtemos A=-1/4 e B=1/4. Concluímos que uma solução particular para (3.434) é

yp⁢(t)=-14⁢cos⁡(t)+14⁢sen⁡(t). (3.440)
