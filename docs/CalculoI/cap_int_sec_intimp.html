4.8 Integrais Impróprias

A integral

∫abf⁢(x)⁢𝑑x (4.527)

é chamada de integral imprópria quando a,b→±∞ ou f tem uma descontinuidade infinita no intervalo [a,b]. Quando a integral existe, dizemos que ela é convergente.

Exemplo 4.8.1.

Estudemos os seguintes casos:

  1. a)

    ∫-2∞1x2⁢𝑑x É uma integral imprópria, pois o intervalo de integração [-2,∞) é infinito.

  2. b)

    ∫-∞1ex⁢𝑑x É uma integral imprópria, pois o intervalo de integração (-∞⁢,1] é infinito.

  3. c)

    ∫-111x+1⁢𝑑x É uma integral imprópria, pois o integrando 1/(x+1) tem uma assíntota vertical no extremo esquerdo do intervalo de integração.

  4. d)

    ∫02x-1x2-1⁢𝑑x Não é uma integral imprópria, pois o integrando é contínuo por partes no intervalo [0,2].

  5. e)

    ∫-22x-1x2-1⁢𝑑x É uma integral imprópria, pois o integrando tem uma descontinuidade infinita em x=-1.

4.8.1 Limites de integração infinitos

No caso de integrais impróprias da forma

∫a∞f⁢(x)⁢𝑑x (4.528)

calculamos

∫a∞f⁢(x)⁢𝑑x=limb→∞⁡∫abf⁢(x)⁢𝑑x. (4.529)
Exemplo 4.8.2.

Vamos calcular

∫1∞1x2⁢𝑑x (4.530)

Temos

∫1∞1x2⁢𝑑x =limb→∞⁡∫1b1x2⁢𝑑x (4.531)
=limb→∞⁡[-1x]1b (4.532)
=limb→∞⁡[-1b0+11] (4.533)
=1 (4.534)

Analogamente, calculamos

∫-∞bf⁢(x)⁢𝑑x=lima→-∞⁡∫abf⁢(x)⁢𝑑x. (4.535)
Exemplo 4.8.3.

Vamos calcular

∫-∞2ex⁢𝑑x. (4.536)

Temos

∫-∞2ex⁢𝑑x =lima→-∞⁡∫a2ex⁢𝑑x (4.537)
=lima→-∞⁡[ex]a2 (4.538)
=lima→-∞⁡e2-ea0 (4.539)
=e2 (4.540)

No caso de integrais impróprias da forma

∫-∞∞f⁢(x)⁢𝑑x (4.541)

escolhemos um c qualquer e calculamos

∫-∞∞f⁢(x)⁢𝑑x=∫-∞cf⁢(x)⁢𝑑x+∫c∞f⁢(x)⁢𝑑x (4.542)

Dizemos que a integral é divergente no caso de ao menos uma das integrais à direita ser divergente.

Exemplo 4.8.4.

Vamos calcular

∫-∞∞2⁢d⁢x1+4⁢x2 (4.543)

Escolhendo c=0, temos

∫-∞∞2⁢d⁢x1+4⁢x2 =∫-∞02⁢d⁢x1+4⁢x2+∫0∞2⁢d⁢x1+4⁢x2 (4.544)
=lima→-∞⁡∫a02⁢d⁢x1+4⁢x2+limb→∞⁡∫0b2⁢d⁢x1+4⁢x2 (4.545)
=lima→-∞⁡[arc⁡tg⁡(2⁢x)]a0+limb→∞⁡[arc⁡tg⁡(2⁢x)]0b (4.546)
=lima→-∞⁡[arc⁡tg⁡(0)-arc⁡tg⁡(a)-π4]a0 (4.547)
+limb→∞⁡[arc⁡tg⁡(b)π4-arc⁡tg⁡(0)]0b (4.548)
=π4+π4=π2 (4.549)

4.8.2 Integrandos com descontinuidade infinita

No caso de integrais impróprias em que o integrando tem descontinuidade infinita no limite de integração inferior, calculamos

∫abf⁢(x)⁢𝑑x=limc→a+⁡∫cbf⁢(x),d⁢x. (4.550)

Se a descontinuidade for no limite superior, então calculamos

∫abf⁢(x)⁢𝑑x=limc→b-⁡∫acf⁢(x),d⁢x. (4.551)
Exemplo 4.8.5.

Vamos calcular

∫122⁢x-2(x-1)2⁢𝑑x (4.552)

Temos

∫12d⁢x(x-1)2⁢𝑑x =limc→1+⁡∫c2d⁢x(x-1)2 (4.553)
=limc→1+⁡∫x=c2d⁢uu2 (4.554)

onde, usamos a substituição u=x-1 e d⁢u=d⁢x. Segue que,

∫12d⁢x(x-1)2⁢𝑑x =limc→1+⁡[-u-1]x=c2 (4.555)
=limc→1+⁡[-1x-1]c2 (4.556)
=limc→1+⁡[-1+1c-10+] (4.557)
=∞ (4.558)

No caso do integrando ter descontinuidade infinita em um ponto interno do limite de integração, calculamos

∫abf⁢(x)⁢𝑑x=∫acf⁢(x)⁢𝑑x+∫cbf⁢(x)⁢𝑑x, (4.559)

onde, x=c é o ponto de descontinuidade de f.

Exemplo 4.8.6.

Vamos calcular

∫03d⁢xx-2 (4.560)

Fazemos

∫03d⁢xx-2 =∫02d⁢xx-2+∫23d⁢xx-2 (4.561)
=limc→2-⁡∫0cd⁢xx-2+limc→2+⁡∫c3d⁢xx-2 (4.562)
=limc→2-⁡[ln⁡|x-2|]0c+limx→2+⁡[ln⁡|x-2|]c3 (4.563)
=limc→2-⁡[ln⁡|c-2|-∞+ln⁡|-2|]0c+limx→2+⁡[ln⁡|1|-ln⁡|c-2|-∞]c3 (4.564)
=-∞+∞ (4.565)

no que concluímos que a integral é divergente.

Exercícios resolvidos

ER 4.8.1.

Para quais valores de p a integral

∫1∞d⁢xxp (4.566)

é convergente.

Solução.

Por definição

∫1∞d⁢xxp=limb→∞⁡∫1bd⁢xxp (4.567)

Para p=1, temos

∫1∞d⁢xx =limb→∞⁡∫1bd⁢xx (4.568)
=limb→∞⁡[ln⁡|x|]1b (4.569)
=limb→∞⁡[ln⁡|b|∞-ln⁡|1|] (4.570)
=∞ (4.571)

Ou seja, para p=1 a integral é divergente. Agora, para p≠1, temos

∫1∞d⁢xxp =limb→∞⁡∫1bd⁢xxp (4.572)
=limb→∞⁡[x1-p1-p]1b (4.573)
=limb→∞⁡[b1-p1-p-11-p] (4.574)

Para p<1, temos que b1-p→∞ quando b→∞. Agora, para p>1, temos que b1-p→0 quando b→∞. Logo, concluímos que a integral é convergente para p>1 e

∫1∞d⁢xxp=1p-1,p>1. (4.575)
ER 4.8.2.

Calcule

∫1∞d⁢x(x-1)2 (4.576)
Solução.

Notamos que, além do limite de integração infinito, o integrando tem uma descontinuidade infinita em x=1. Logo, calculamos

∫1∞d⁢xx-1 =limb→∞⁡∫1bd⁢x(x-1)2 (4.577)
=limb→∞⁡[lima→1+⁡∫abd⁢x(x-1)2] (4.578)
=limb→∞⁡[lima→1+⁡[11-x]ab] (4.579)
=limb→∞⁡lima→1+⁡(11-b0-11-a-∞) (4.580)
=+∞ (4.581)
