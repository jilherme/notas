5.1 Integrais indefinidas

Seja f⁢(x) uma função. Dizemos que F⁢(x) é uma primitiva de f⁢(x) quando

F′⁢(x)=f⁢(x) (5.1)

para todo x no domínio da f.

Exemplo 5.1.1.

Seja f⁢(x)=2⁢x. Notemos que F⁢(x)=x2 é uma primitiva de f, pois

F′⁢(x)=2⁢x=f⁢(x). (5.2)

Também, se C é uma constante qualquer, F⁢(x)=x2+C é primitiva de f⁢(x). De fato, lembrando que a derivada de uma constante é zero, temos

F′⁢(x)=2⁢x+0=2⁢x=f⁢(x). (5.3)
Observação 5.1.1.

Se F⁢(x) é uma primitiva de f⁢(x), então F⁢(x)+C, com C constante, também é primitiva de f⁢(x).

A integral indefinida de uma função f⁢(x) é denotada por

∫f⁢(x)⁢𝑑x (5.4)

e é definida por

∫f⁢(x)⁢𝑑x=F⁢(x)+C, (5.5)

onde F⁢(x) é uma primitiva de f⁢(x).

Exemplo 5.1.2.

Do observado no exemplo anterior (Exemplo 5.1.1), temos

∫2⁢x⁢𝑑x=x2+C. (5.6)

Com o Sympy, podemos computar a integral indefinida acima com o comando8686Veja a Observação 5.0.1.:

integrate(2*x)

Observe que o Sympy não adiciona a constante indeterminada.

5.1.1 Regras básicas de integração

Das regras básicas de derivação, podemos inferir as seguintes regras para integração:

  • •

    ∫0⁢𝑑x=C.

  • •

    ∫1⁢𝑑x=x+C.

  • •

    ∫a⁢f⁢(x)⁢𝑑x=a⁢∫f⁢(x)⁢𝑑x, a≠0 constante.

  • •

    ∫f⁢(x)±g⁢(x)⁢d⁢x=∫f⁢(x)⁢𝑑x±∫g⁢(x)⁢𝑑x.

Exemplo 5.1.3.
∫x-3⁢x2⁢d⁢x =∫x⁢𝑑x-∫3⁢x2⁢𝑑x (5.7)
=∫22⁢x⁢𝑑x-x3+C (5.8)
=12⁢∫2⁢x⁢𝑑x-x3+C (5.9)
=x22-x3+C. (5.10)

No exemplo anterior (Exemplo 5.1.3) integramos funções potências. Da regra de derivação

[xn]′=n⁢xn-1,n≠0, (5.11)

temos

∫xn⁢𝑑x =∫n+1n+1⁢xn⁢𝑑x,n≠-1, (5.12)
=1n+1⁢∫(n+1)⁢xn⁢𝑑x, (5.13)
=xn+1n+1+C. (5.14)

Ou seja, obtemos a seguinte regra de integração para função potência

∫xn⁢𝑑x=xn+1n+1+C,n≠-1. (5.15)
Exemplo 5.1.4.
∫3⁢x-2⁢𝑑x=3⁢∫x-2⁢𝑑x=3⁢x-2+1-2+1+C=-3⁢x-1+C=-3x+C. (5.16)

Das regras de derivação para a função exponencial natural e logaritmo natural, temos

  • •

    ∫ex⁢𝑑x=ex+C.

  • •

    ∫1x⁢𝑑x=ln⁡x+C.

Também, como (ax)′=ax⁢ln⁡a, temos

∫ax⁢𝑑x=axln⁡a+C. (5.17)
