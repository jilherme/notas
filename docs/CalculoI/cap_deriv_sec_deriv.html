3.2 Função derivada

A derivada de uma função f em relação à variável x é a função f′=d⁢fd⁢x cujo valor em x é

limh→0⁡f⁢(x+h)-f⁢(x)h, (3.19)

quando este limite existe. Dizemos que f é derivável (ou diferenciável) em um ponto x de seu domínio, quando o limite dado em (3.19) existe. Se isso ocorre para todo número real x, dizemos que f é derivável em toda parte.

Exemplo 3.2.1.

A derivada de f⁢(x)=x2 é

f′⁢(x) =limh→0⁡f⁢(x+h)-f⁢(x)h (3.20)
=limh→0⁡(x+h)2-x2h (3.21)
=limh→0⁡x2+2⁢x⁢h+h2-x2h (3.22)
=limh→0⁡2⁢x+h=2⁢x. (3.23)

Com o SymPy, podemos usar os seguintes comandos para verificarmos este resultado:

h = symbols(’h’,real=True)
f = lambda x: x**2
limit((f(x+h)-f(x))/h,h,0)
Observação 3.2.1.

A derivada à direita (à esquerda) de uma função f em um ponto x é definida por

f±′⁢(x)=d⁢fd⁢x±=limh→0±⁡f⁢(x+h)-f⁢(x)h. (3.24)

Desta forma, no caso de pontos extremos do domínio de uma função, empregamos a derivada lateral correspondente.

Exemplo 3.2.2.

A função valor absoluto é derivável para todo x≠0 e não é derivável em x=0. De fato, para x<0 temos

f′⁢(x) =limx→0⁡|x+h|-|x|h (3.25)
=limh→0⁡-(x+h)+xh (3.26)
=limh→0⁡hh=1. (3.27)

Analogamente, para x>0 temos

f′⁢(x) =limx→0⁡|x+h|-|x|h (3.28)
=limx→0⁡x+h-xh (3.29)
=limx→0⁡hh=1. (3.30)

Agora, para x=0, devemos verificar as derivadas laterais:

f+′⁢(0) =limh→0+⁡|h|-|0|h=limh→0+⁡hh=1, (3.31)
f-′⁢(0) =limh→0-⁡|h|-|0|h=limh→0-⁡-hh=-1. (3.32)

Como as derivadas laterais são diferentes, temos que y=|x| não é derivável em x=0.

Exemplo 3.2.3.

Vamos calcular a derivada de f⁢(x)=x. Para x=0, só faz sentido calcular a derivada lateral à direta:

f′⁢(0)=limh→0+⁡h-0h=+∞. (3.33)

Ou seja, f⁢(x)=x não é derivável em x=0. Agora, para x>0, temos

f′⁢(x) =limh→0⁡x+h-xh (3.34)
=limh→0⁡x+h-xh⁢x+h+xx+h+x (3.35)
=limh→0⁡x+h-xh⁢(x+h+x) (3.36)
=12⁢x. (3.37)
