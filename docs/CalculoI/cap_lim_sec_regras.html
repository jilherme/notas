3.2 Regras para o cálculo de limites

3.2.1 Regras de cálculo

    Assista no YouTube! Assista no archive.org! Escute a audioleitura!

Sejam dados os seguintes limites

limx→x0⁡f⁢(x)=L1 (3.10)
limx→x0⁡g⁢(x)=L2 (3.11)

com x0,L1,L2 números reais. Então, valem as seguintes regras:

  • •

    Regra da multiplicação por um escalar:

    limx→x0⁡k⋅f⁢(x) =k⋅limx→x0⁡f⁢(x) (3.12)
    =k⋅L1, (3.13)

    para qualquer número real k.

  • •

    Regra da soma/subtração:

    limx→x0⁡f⁢(x)±g⁢(x) =limx→x0⁡f⁢(x)±limx→x0⁡g⁢(x) (3.14)
    =L1±L2 (3.15)
  • •

    Regra do produto:

    limx→x0⁡f⁢(x)⋅g⁢(x) =limx→x0⁡f⁢(x)⋅limx→x0⁡g⁢(x) (3.16)
    =L1⋅L2 (3.17)
  • •

    Regra do quociente:

    limx→x0⁡f⁢(x)g⁢(x) =limx→x0⁡f⁢(x)limx→x0⁡g⁢(x) (3.18)
    =L1L2,L2≠0 (3.19)
  • •

    Regra da potenciação:

    limx→x0⁡(f⁢(x))s =(limx→x0⁡f⁢(x))s (3.20)
    =L1s,L1s∈ℝ (3.21)

Podemos usar essas regras para calcularmos limites.

Exemplo 3.2.1.

Consideremos os seguintes casos:

  1. a)

    limx→-1⁡2⁢x

    limx→-1⁡2⁢x =2⁢limx→-1⁡x (3.22)
    =2⋅(-1)=-2 (3.23)

    Com o Python+SymPy, podemos computar este limite com

    ⬇
    1     from sympy import *
    2     x = Symbol("x")
    3     limit(2*x,x,-1)
  2. b)

    limx→2⁡x2-1

    limx→2⁡x2-1 =limx→2⁡x2-limx→2⁡1 (3.24)
    =(limx→2⁡x)2-1 (3.25)
    =22-1=3. (3.26)

    Com o Python+SymPy, podemos computar este limite com os seguintes comandos.

    ⬇
    1     from sympy import *
    2     x = Symbol("x")
    3     limit(x**2-1,x,2)
  3. c)

    limx→0⁡1-x2.

    limx→0⁡1-x2 =limx→0⁡1-x2 (3.27)
    =limx→0⁡1-(limx→0⁡x)2 (3.28)
    =1-(0)2 (3.29)
    =1. (3.30)

    Com o Python+SymPy, podemos computar este limite com

    ⬇
    1     from sympy import *
    2     x = Symbol("x")
    3     limit(sqrt(1-x**2),x,0)
  4. d)

    limx→0⁡(x2-1)⁢(x-2)(x-1)⁢(x-2)

    limx→0⁡(x2-1)⁢(x-2)(x-1)⁢(x-2) =limx→0⁡[(x2-1)⋅(x-2)]limx→0⁡[(x-1)⋅(x-2)] (3.31)
    =limx→0⁡(x2-1)⋅limx→0⁡(x-2)limx→0⁡(x-1)⋅limx→0⁡(x-2) (3.32)
    =22=1. (3.33)
Proposição 3.2.1.

(Limites de polinômios) Se

p⁢(x)=an⁢xn+an-1⁢xn-1+⋯+a0, (3.34)

então

limx→b⁡p⁢(x) =p⁢(b) (3.35)
=an⁢bn+an-1⁢bn-1+⋯+a0, (3.36)

para qualquer dado número real b.

Demonstração.

Segue das regras da soma, da multiplicação por escalar e da potenciação.

limx→b⁡p⁢(x) =limx→b⁡an⁢xn+an-1⁢xn-1+⋯+a0 (3.37)
=limx→b⁡an⁢xn+limx→b⁡an-1⁢xn-1+⋯+limx→b⁡a0 (3.38)
=an⁢(limx→b⁡x)n+an-1⁢(limx→b⁡x)n-1+⋯+a0 (3.39)
=an⁢bn+an-1⁢bn-1+⋯+a0=p⁢(b). (3.40)

∎

Exemplo 3.2.2.
limx→2⁡2⁢x4-2⁢x2+x =2⁢(2)4-2⁢(2)2+2 (3.41)
=4+2. (3.42)

Com o Python+SymPy, podemos computar este limite com os seguintes comandos.

⬇
1     from sympy import *
2     x = Symbol("x")
3     limit(2*x**4-2*x**2+x,x,sqrt(2))
Proposição 3.2.2.

(Limite de funções racionais) Sejam r⁢(x)=p⁢(x)/q⁢(x) uma função racional e b um número real tal que q⁢(b)≠0. Então,

limx→b⁡p⁢(x)q⁢(x)=p⁢(b)q⁢(b). (3.43)
Demonstração.

Segue da regra do limite do quociente e da Proposição 3.2.1.

limx→b⁡p⁢(x)q⁢(x) =limx→b⁡p⁢(x)limx→b⁡q⁢(x) (3.44)
=p⁢(b)q⁢(b). (3.45)

∎

Exemplo 3.2.3.
limx→0⁡(x2-1)⁢(x-2)(x-1)⁢(x-2) =(02-1)⁢(0-2)(0-1)⁢(0-2) (3.46)
=22=1. (3.47)

Com o Python+SymPy, podemos computar este limite com os comandos.

⬇
1     from sympy import *
2     x = Symbol("x")
3     limit((x**2-1)*(x-2)/((x-1)*(x-2)),x,0)

3.2.2 Indeterminação 0/0

    Assista no YouTube! Assista no archive.org! Escute a audioleitura!

Quando limx→a⁡f⁢(a)=0 e limx→a⁡g⁢(a)=0, dizemos que

limx→a⁡f⁢(x)g⁢(x) (3.48)

é uma indeterminação do tipo 0/0. Em vários destes casos, podemos calcular o limite eliminando o fator em comum (x-a).

Exemplo 3.2.4.
limx→2⁡(x2-1)⁢(x-2)(x-1)⁢(x-2) =limx→2⁡x2-1x-1 (3.49)
=22-12-1=3. (3.50)

Com o Python+SymPy, podemos computar o limite acima com os seguintes comandos.

⬇
1     from sympy import *
2     x = Symbol("x")
3     limit((x**2-1)*(x-2)/((x-1)*(x-2)),x,2)

Quando o fator em comum não aparece explicitamente, podemos tentar trabalhar algebricamente de forma a explicitá-lo.

Exemplo 3.2.5.

No caso do limite

limx→1⁡x3-3⁢x2-x+3x2+x-2 (3.51)

temos que o denominador p⁢(x)=x3-3⁢x2-x+3 se anula em x=1, assim como o denominador q⁢(x)=x2+x-2. Assim sendo, (x-1) é um fator comum entre p⁢(x) e q⁢(x). Para explicitá-lo, calculamos

p⁢(x)x-1 =x3-3⁢x2-x+3x-1 (3.52)
=x2-2⁢x-3 (3.53)

e

q⁢(x)x-1 =x2+x-2x-1 (3.54)
=x+2. (3.55)

Com o Python+SymPy, podemos computar estas divisões com os seguintes comandos.

⬇
1     from sympy import *
2     x = Symbol("x")
3     simplify((x**3-3*x**2-x+3)/(x-1))
4     simplify((x**2+x-2)/(x-1))

Realizadas as divisões, temos

p⁢(x)=(x-1)⁢(x2-2⁢x-3) (3.56)

e

q⁢(x)=(x-1)⁢(x+2). (3.57)

Com isso, segue que

limx→1⁡x3-3⁢x2-x+3x2+x-2 =limx→1⁡(x-1)⁢(x2-2⁢x-3)(x-1)⁢(x+2) (3.58)
=limx→1⁡x2-2⁢x-3x+2=-43. (3.59)

Use Python+SymPy para computar este limite!

Exemplo 3.2.6.

No caso de

limx→0⁡1-x-1x (3.60)

temos uma indeterminação do tipo 0/0 envolvendo uma raiz. Neste caso, podemos calcular o limite usando de racionalização.

limx→0⁡1-x-1x =limx→0⁡1-x-1x⁢1-x+11-x+1 (3.61)
=limx→0⁡1-x-1x⁢(1-x+1) (3.62)
-limx→0⁡-xx⁢(1-x+1) (3.63)
=limx→0⁡-11-x+1=-12. (3.64)

Verifique computando com o Python+SymPy.

Exercícios resolvidos

ER 3.2.1.

Calcule

limx→-1⁡x-x2x2+3. (3.65)
Solução.

Usando das propriedades de limites, calculamos

limx→-1⁡x-x2x2+3 =limx→-1⁡x-x2limx→-1⁡x2+3 (3.66)
=-1-(-1)2limx→-1⁡x2+3 (3.67)
=-24 (3.68)
=-1. (3.69)
ER 3.2.2.

Assumindo que o limx→2⁡f⁢(x)=L e que

limx→2⁡f⁢(x)-2x+2=1, (3.70)

forneça o valor de L.

Solução.

Das propriedades de limites, temos

limx→2⁡f⁢(x)-2x+2=1 (3.71)
limx→2⁡f⁢(x)-2limx→2⁡x+2=1 (3.72)
limx→2⁡f⁢(x)-limx→2⁡22+2=1 (3.73)
L-24=1 (3.74)
L-2=4 (3.75)
L=6. (3.76)
ER 3.2.3.

Calcule

limx→-1⁡x+12-x2+3. (3.77)
Solução.

Neste caso, não podemos usar a regra do quociente, pois

limx→-1⁡2-x2+3=0. (3.78)

Agora, como também temos

limx→-1⁡x+1=0, (3.79)

concluímos se tratar de uma indeterminação 0/0. Por racionalização, obtemos

limx→-1⁡x+12-x2+3 =limx→-1⁡x+12-x2+3⁢2+x2+32+x2+3 (3.80)
=limx→-1⁡(x+1)⁢(2+x2+3)4-(x2+3) (3.81)
=limx→-1⁡(x+1)⁢(2+x2+3)1-x2 (3.82)
=limx→-1⁡(x+1)⁢(2+x2+3)(1+x)⁢(1-x) (3.83)
=limx→-1⁡2+x2+31-x (3.84)
=42=2. (3.85)
