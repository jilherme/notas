3.3 Regras básicas de derivação

Vejamos as derivadas da função constante e da função potência.

  • •

    d⁢kd⁢x=0, onde k é uma constante.

    Dem.: Com f⁢(x)≡k temos

    d⁢kd⁢x =limh→0⁡f⁢(x+h)-f⁢(x)h (3.38)
    =limh→0⁡k-kh (3.39)
    =limh→0⁡0=0. (3.40)

    No SymPy, podemos usar os seguintes comandos para obtermos tal regra de derivação:

    k = symbols(’k’,real=True)
diff(k,x)
  • •

    d⁢xnd⁢x=n⁢xn-1, para n inteiro positivo.

    Dem.: Com f⁢(x)=xn, temos

    d⁢xnd⁢x =limh→0⁡f⁢(x+h)-f⁢(x)h (3.41)
    =limh→0⁡(x+h)n-xnh (3.42)
    =limh→0⁡xn+n⁢xn-1⁢h+n⁢(n-1)2⁢xn-2⁢h2+⋯+hn-xnh (3.43)
    =limh→0⁡n⁢xn-1+n⁢(n-1)2⁢xn-2⁢h+⋯+hn-1 (3.44)
    =n⁢xn-1. (3.45)

    No SymPy, podemos usar os seguintes comandos para obtermos tal regra de derivação:

    n = symbols(’n’,integer=True, positive=True)
simplify(diff(x**n,x))
Exemplo 3.3.1.

Vejamos os seguintes casos:

  1. a)

    d⁢2d⁢x=0.

  2. b)

    d⁢x3d⁢x=3⁢x2.

3.3.1 Multiplicação por constante e soma

Sejam c um número real, u e v funções. Temos as seguintes regras básicas de derivação:

  • •

    (c⁢u)′=c⁢u′.

    Dem.:

    dd⁢x⁢(c⁢u)⁢(x) =limh→0⁡c⁢u⁢(x+h)-c⁢u⁢(x)h (3.46)
    =c⁢limh→0⁡u⁢(x+h)-u⁢(x)h (3.47)
    =c⁢d⁢ud⁢x. (3.48)

    No SymPy, podemos usar os seguintes comandos para obtermos tal regra de derivação:

    c = symbols(’c’, real=True)
u = Function(’u’, real=True)(x)
diff(c*u,x)
  • •

    (u±v)′=u′±v′.

    Dem.:

    (u±v)′⁢(x) =limh→0⁡(u±v)⁢(x+h)-(u±v)⁢(x)h (3.49)
    =limh→0⁡[u⁢(x+h)-u⁢(x)h±v⁢(x+h)-v⁢(x)h] (3.50)
    =u′⁢(x)±v′⁢(x). (3.51)

    No SymPy, podemos usar os seguintes comandos para obtermos a regra de derivação para soma:

    u = Function(’u’, real=True)(x)
v = Function(’v’, real=True)(x)
diff(u+v,x)
Exemplo 3.3.2.
dd⁢x⁢(x3-2⁢x-1) =dd⁢x⁢x3-2⁢d⁢xd⁢x-d1d⁢x (3.52)
=3⁢x2-2. (3.53)

3.3.2 Produto e quocientes

Sejam y=u⁢(x) e y=v⁢(x) funções deriváveis, com v⁢(x)≠0. Então:

  • •

    (u⁢v)′=u′⁢v+u⁢v′.

    Dem.:

    (u⁢v)′⁢(x) =limh→0⁡(u⁢v)⁢(x+h)-(u⁢v)⁢(x)h (3.54)
    =limh→0⁡u⁢(x+h)⁢v⁢(x+h)-u⁢(x)⁢v⁢(x)h (3.55)
    =limh→0[u⁢(x+h)⁢v⁢(x+h)-u⁢(x)⁢v⁢(x+h)h (3.56)
       +u⁢(x)⁢v⁢(x+h)-u⁢(x)⁢v⁢(x)h] (3.57)
    =limh→0⁡u⁢(x+h)-u⁢(x)h⁢v⁢(v+h) (3.58)
    +limh→0⁡u⁢(x)⁢v⁢(x+h)-v⁢(x)h (3.59)
    =u′⁢(x)⁢v⁢(x)+u⁢(x)⁢v′⁢(x). (3.60)

    No SymPy, podemos usar os seguintes comandos para obtermos tal regra de derivação:

    u = Function(’u’, real=True)(x)
v = Function(’v’, real=True)(x)
diff(u*v,x)
  • •

    (uv)′=u′⁢v-u⁢v′v2.

    Dem.:

    (uv)′⁢(x) =limh→0⁡(uv)⁢(x+h)-(uv)⁢(x)h (3.61)
    =limh→0⁡u⁢(x+h)⁢v⁢(x)-u⁢(x)⁢v⁢(x+h)v⁢(x+h)⁢v⁢(x)h (3.62)
    =limh→0[u⁢(x+h)⁢v⁢(x)-u⁢(x)⁢v⁢(x)h (3.63)
       -u⁢(x)⁢v⁢(x+h)-u⁢(x)⁢v⁢(x)h]1v⁢(x)⁢v⁢(x+h) (3.64)
    =[limh→0u⁢(x+h)-u⁢(x)hv(x) (3.65)
    -limh→0u(x)v⁢(x+h)-v⁢(x)h]limh→01v⁢(x)⁢v⁢(x+h) (3.66)
    =u′⁢(x)⁢v⁢(x)-u⁢(x)⁢v′⁢(x)v2⁢(x). (3.67)

    No SymPy, podemos usar os seguintes comandos para obtermos tal regra de derivação:

    u = Function(’u’, real=True)(x)
v = Function(’v’, real=True)(x)
simplify(diff(u/v,x))
Exemplo 3.3.3.

Vejamos os seguintes casos:

  1. a)
    dd⁢x⁢[(x2+x)⁢(1+x3)] =[dd⁢x⁢(x2+x)]⁢(1+x3) (3.68)
    +(x2+x)⁢[dd⁢x⁢(1+x3)] (3.69)
    =(2⁢x+1)⁢(1+x3)+(x2+x)⁢3⁢x2 (3.70)
    =2⁢x+2⁢x4+1+x3+3⁢x4+3⁢x3 (3.71)
    =5⁢x4+4⁢x3+2⁢x+1. (3.72)

    Com o SymPy, podemos computar esta derivada com o seguinte comando1313Veja a Observação 3.0.1.:

    d = diff((x**2+x)*(1+x**3),x)
simplify(d)
  2. b)
    dd⁢x⁢(x2+x1-x3) =[dd⁢x⁢(x2+x)]⁢(1-x3)-(x2+x)⁢[dd⁢x⁢(1-x3)](1-x3)2 (3.73)
    =(2⁢x+1)⁢(1-x3)+(x2+x)⁢3⁢x21-2⁢x3+x6 (3.74)
    =2⁢x-2⁢x4+1-x3+3⁢x4+3⁢x31-2⁢x3+x6 (3.75)
    =x4+2⁢x3+2⁢x+1x6-2⁢x3+1 (3.76)

    Com o SymPy, podemos computar esta derivada com o seguinte comando1414Veja a Observação 3.0.1.:

    d = diff((x**2+x)/(1-x**3),x)
simplify(d)
Observação 3.3.1.

(Derivada de funções potência) No início desta seção, vimos que

d⁢xnd⁢x=n⁢xn-1, (3.77)

para n>0 inteiro. Agora, podemos afirmar que este é, também, o caso quando n<0 inteiro. De fato, se n<0, então

(xn)′ =(1x-n) (3.78)
=(1)′⁢x-n-1⋅(x-n)′(x-n)2 (3.79)
=0-(-n)⁢x-n-1x-2⁢n (3.80)
=n⁢x2⁢n-n-1=n⁢xn-1. (3.81)

O SymPy, obtém o mesmo resultado, verifique usando os comandos:

var(’n’,integer=True)
simplify(diff(x**n,x))

dd⁢x⁢x-5=-5⁢x-5-1=-5⁢x-4. (3.82)

3.3.3 Derivadas de funções exponenciais

Seja f⁢(x)=ax, a>0 e a≠1. Então

f′⁢(x) =limh→0⁡f⁢(x+h)-f⁢(x)h (3.83)
=limh→0⁡ax+h-axh (3.84)
=limh→0⁡ax⁢ah-axh (3.85)
=ax⁢limh→0⁡ah-1h (3.86)

Pode-se mostrar que

limh→0⁡ah-1h=ln⁡a. (3.87)

Desta forma, temos

d⁢axd⁢x=ax⁢ln⁡a. (3.88)

Para a função exponencial natural y=ex, temos

d⁢exd⁢x =ex⁢ln⁡e (3.89)
=ex. (3.90)
Exemplo 3.3.5.

Vejamos os seguintes casos:

  1. a)

    d2xd⁢x=2x⁢ln⁡2.

  2. b)
    d⁢e2⁢xd⁢x =dd⁢x⁢(ex⁢ex) (3.91)
    =ex⁢ex+ex⁢ex (3.92)
    =2⁢e2⁢x. (3.93)

3.3.4 Derivadas de ordens mais altas

A derivada de uma função y=f⁢(x) em relação a x é a função y=f′⁢(x). Quando esta é diferenciável, podemos calcular a derivada da derivada. Esta é conhecida como a segunda derivada de f, denotamos

f′′⁢(x):=(f′⁢(x))′⁢ou⁢d2d⁢x⁢f⁢(x)=dd⁢x⁢(dd⁢x⁢f⁢(x)). (3.94)
Exemplo 3.3.6.

Seja f⁢(x)=1/x. Então, a primeira derivada de f é

f′⁢(x)=[x-1]′=-x-2. (3.95)

De posse da primeira derivada, podemos calculamos a segunda derivada de f, como segue:

f′′⁢(x)=[f′⁢(x)]′=[-x-2]′=2⁢x-3. (3.96)

Generalizando, quando existe, a n-ésima derivada de uma função y=f⁢(x), n≥1, é recursivamente definida (e denotada) por

f(n)⁢(x):=[f(n-1)]′⁢ou⁢dnd⁢xn⁢f⁢(x):=dd⁢x⁢[dn-1d⁢xn-1⁢f⁢(x)], (3.97)

com f(3)≡f′′′, f(2)≡f′′, f(1)≡f′ e f(0)≡f.

Exemplo 3.3.7.
  1. a)

    Para obtermos a terceira derivada de f⁢(x)=x3-2⁢x, calculamos

    f′⁢(x)=3⁢x2-2. (3.98)

    então

    f′′⁢(x)=6⁢x (3.99)

    e, finalmente,

    f′′′⁢(x)=6. (3.100)

    Com o Sympy, podemos computar esta derivada com o seguinte comando:

    diff(x**3-2*x,x,3)
  2. b)

    A milionésima primeira derivada de f⁢(x)=ex é f(1001)⁢(x)=ex. Verifique!
