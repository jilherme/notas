3.5 Limites infinitos

O limite de uma função nem sempre existe. Entretanto, em muitos destes casos, podemos concluir mais sobre a tendência da função. Por exemplo, dizemos que o limite de uma dada função f⁢(x) é infinito quando x tende a um número x0, quando f⁢(x) torna-se arbitrariamente grande para todos os valores de x suficientemente próximos de x0, mas x≠0. Neste caso, escrevemos

limx→x0⁡f⁢(x)=∞. (3.198)

A Figura 3.15, é uma ilustração de f⁢(x)→∞ quando x→x0.

Exemplo 3.5.1.

Vejamos o caso de

limx→0⁡1x2. (3.199)

Ao tomarmos x próximo de x0=0, obtemos os seguintes valores de f⁢(x):

x -10-1 -10-2 -10-3 →0← 10-3 10-2 10-1
f(x) -102 -104 -106 →∞← 106 104 102

Veja o esboço do gráfico de f⁢(x) na Figura 3.16.

Podemos concluir que os valores de f⁢(x) podem ser tomados arbitrariamente grandes ao escolhermos qualquer x suficientemente próximo de 0, com x≠0. I.e.,

limx→0⁡1x2=∞. (3.200)

Com o Python+SymPy, podemos computar este limite com o seguinte comando:

⬇
1     from sympy import *
2     limit(1/x**2,x,0)

Atenção! Na verdade, este comando computa o limite lateral à direita. Na sequência, discutimos sobre limites laterais infinitos.

Definimos os limites laterais infinitos

limx→x0-⁡f⁢(x)=∞ e limx→x0+⁡f⁢(x)=∞. (3.201)

No primeiro caso, os valores de f⁢(x) são arbitrariamente grandes conforme os valores de x→x0 e x<x0. No segundo caso, os valores de f⁢(x) são arbitrariamente grandes conforme os valores de x→x0 e x>x0.

Exemplo 3.5.2.
limx→1+⁡1x-1=∞. (3.202)

De fato, conforme tomamos valores de x próximos de 1, com x>1, os valores de f⁢(x)=1/(x-1) tornam-se cada vez maiores. Veja o esboço do gráfico de f⁢(x) na Figura 3.17.

Com o Python+SymPy, podemos computar este limite com o seguinte comando:

⬇
1     from sympy import *
2     limit(1/(x-1),x,0,'+')

Analogamente a definição de limite infinito, dizemos que o limite de uma dada função f⁢(x) é menos infinito quando x tende a x0, quando f⁢(x) torna-se arbitrariamente pequeno para valores de x suficientemente próximos de x0, com x≠x0. Neste caso, escrevemos

limx→x0⁡f⁢(x)=-∞. (3.203)

De forma similar, definimos os limites laterais f⁢(x)→-∞ quando x→x0±.

Exemplo 3.5.3.

Observe que

∄⁢limx→0⁡1x (3.204)

e que não podemos concluir que este limite é ∞ ou -∞. Isto ocorre, pois

limx→0-⁡1x=-∞ e limx→0+⁡1x=+∞. (3.205)
Exemplo 3.5.4.
limx→-1⁡-1(x+1)2=-∞. (3.206)

De fato, podemos inferir este limite a partir do gráfico da função f⁢(x)=1/(x+1)2. Este é uma translação de uma unidade à esquerda do gráfico de y=1/x2, seguida de uma reflexão em torno de eixo x. Veja a Figura 3.18.

Com o Python+SymPy, podemos computar este limite com o seguinte comando:

⬇
1     from sympy import *
2     limit(-1/(x+1)**2,x,-1)

Novamente, observamos que este comando computa apenas o limite lateral à direita.

3.5.1 Assíntotas verticais

Uma reta x=x0 é uma assíntota vertical do gráfico de uma função y=f⁢(x) se

limx→x0-⁡f⁢(x)=±∞ ou limx→x0+⁡f⁢(x)=±∞. (3.207)
Exemplo 3.5.5.

O gráfico da função f⁢(x)=-1/|x| tem uma assíntota vertical em x=0, pois

limx→0⁡-1|x|=-∞. (3.208)

Veja o esboço de seu gráfico na Figura 3.19.

Exemplo 3.5.6.

A função f⁢(x)=x3+2⁢x2-4⁢x-8x2-1 não está definida para valores de x tais que seu denominador se anule, i.e.

x2-1=0⇒x0=-1 ou x1=1. (3.209)

Nestes pontos o gráfico de f pode ter assíntotas verticais. De fato, temos

limx→-1+⁡x3+2⁢x2-4⁢x-8-3x2-10- =+∞, (3.210)
limx→-1-⁡x3+2⁢x2-4⁢x-8-3x2-10+ =-∞, (3.211)

e, também, temos

limx→1+⁡x3+2⁢x2-4⁢x-8-9x2-10+ =-∞, (3.212)
limx→1-⁡x3+2⁢x2-4⁢x-8-9x2-10- =+∞. (3.213)

Com isso, temos que as retas x=-1 e x=1 são assíntotas verticais ao gráfico da função f. Veja a Figura 3.20 para o esboço do gráfico desta função.

Exemplo 3.5.7.

(Função logarítmica) A função logarítmica natural y=ln⁡x é tal que

limx→0+⁡ln⁡x=-∞ (3.214)

i.e., x=0 é uma assíntota vertical ao gráfico de ln⁡x. Isto decorre do fato de y=ln⁡x ser a função inversa de y=ex e, esta, ter uma assíntota horizontal y=055 5 Veja o Exemplo 3.4.6.. A Figura 3.21 é um esboço do gráfico da função ln⁡x.

Exemplo 3.5.8.

As funções trigonométricas y=sec⁡x e y=tg⁡x têm assíntotas verticais x=(2⁢k+1)⁢π2 para k inteiro. Veja as Figuras LABEL:fig:co_tg_graficos.

Exemplo 3.5.9.

As funções trigonométricas y=cosec⁡x e y=cotg⁡x têm assíntotas verticais x=k⁢π para k inteiro. Veja as Figuras LABEL:fig:co_sec_graficos.

3.5.2 Assíntotas oblíquas

Além de assíntotas horizontais e verticais, gráficos de funções podem ter assintota oblíquas. Isto ocorre, particularmente, para funções racionais cujo grau do numerador é maior que o do denominador.

Exemplo 3.5.10.

Consideremos a função racional

f⁢(x)=x2-15⁢x-4. (3.215)

Para buscarmos determinar a assíntota oblíqua desta função, dividimos o numerador pelo denominador, de forma a obtermos

f⁢(x)=(x5+425)⏟quociente+-9255⁢x-4⏟resto. (3.216)

Observamos, agora, que o resto tende a zero quando x→±∞, i.e. f⁢(x)→x5+425 quando x→±∞. Com isso, concluímos que y=x5+425 é uma assíntota oblíqua ao gráfico de f⁢(x). Veja a Figura 3.22.

Observação 3.5.1.

Analogamente à assintotas oblíquas, podemos ter outros tipos de assíntotas determinadas por funções de diversos tipos, por exemplo, assíntotas quadráticas.

3.5.3 Limites infinitos no infinito

Escrevemos

limx→∞⁡f⁢(x)=∞, (3.217)

quando os valores da função f são arbitrariamente grandes para todos os valores de x suficientemente grandes. De forma análoga, definimos

limx→-∞⁡f⁢(x)=∞,limx→∞⁡f⁢(x)=-∞ e limx→-∞⁡f⁢(x)=-∞. (3.218)
Exemplo 3.5.11.

Vejamos os seguintes casos:

  1. a)

    limx→∞⁡x2=∞

  2. b)

    limx→-∞⁡x2=∞

  3. c)

    limx→-∞⁡x3=-∞

  4. d)

    limx→∞⁡ex=∞

  5. e)

    limx→∞⁡ln⁡x=∞

  6. f)

    limx→-∞⁡e-x=∞

Exemplo 3.5.12.
limx→∞⁡x3-10⁢x2+300 =limx→∞⁡x3-10⁢x2+3001⋅1x31x3 (3.219)
=limx→∞⁡1-10x0++300x30+1x30+=∞. (3.220)
Proposição 3.5.1.

Dado um polinômio p⁢(x)=an⁢xn+an-1⁢xn-1+⋯+a0, temos

limx→±∞⁡p⁢(x)=limx→±⁡an⁢xn. (3.221)
Exemplo 3.5.13.

Retornando ao exemplo anterior (Exemplo 3.5.12, temos

limx→∞⁡x3-10⁢x2+300 =limx→∞⁡x3=∞. (3.222)

Exercícios resolvidos

ER 3.5.1.

Calcule

limx→1-⁡x-21-x. (3.223)
Solução.

Temos

limx→1-⁡x-2-11-x0+=-∞. (3.224)

Outra forma de calcular este limite é observar que y=1-x→0+ quando x→1-. Assim, fazendo a mudança de variável y=x-1, temos

limx→1-⁡x-21-x=limy→0+⁡y+1-2y=limy→0+⁡y-1y=-∞. (3.225)

Podemos usar o seguinte comandoPython+SymPy para computar este limite:

⬇
1     from sympy import *
2     limit((x-2)/(1-x),x,1,'-')
ER 3.5.2.

Calcule

limx→1⁡ln⁡|x-1|. (3.226)
Solução.

Começamos observando que

ln⁡|x-1|={ln⁡(1-x),x<1,ln⁡(x-1),x>1. (3.227)

Então, calculando o limite lateral à esquerda, temos

limx→1-⁡ln⁡|x-1| =limx→1-⁡ln⁡(1-x)
=limy→0+⁡ln⁡y=-∞.

Por outro lado, temos

limx→1+⁡ln⁡|x-1| =limx→1+⁡ln⁡(x-1)
=limy→0+⁡ln⁡y=-∞.

Portanto, concluímos que

limx→1⁡ln⁡|x-1|=-∞. (3.228)

Podemos usar os seguintes comandos Python+SymPy para computar os limites laterais:

⬇
1     from sympy import *
2     limit(log(abs(x-1)),x,1,'-')
3     limit(log(abs(x-1)),x,1,'+')

ER 3.5.4.

Calcule

limx→∞⁡e1-x2. (3.231)
Solução.

Observamos que 1-x2→-∞ quando x→∞. Desta forma, fazendo a mudança de variáveis y=1-x2, temos

limx→∞⁡e1-x2=limy→-∞⁡ey=0. (3.232)
