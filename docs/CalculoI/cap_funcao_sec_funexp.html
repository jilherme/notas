1.6 Funções exponenciais

Uma função exponencial tem a forma

f⁢(x)=ax, (1.37)

onde a≠1 é uma constante positiva e é chamada de base da função exponencial.

Funções exponenciais estão definidas em toda parte e têm imagem (0,∞). O gráfico de uma função exponencial sempre contém os pontos (-1,1/a), (0,1) e (1,a). Veja a Figura 1.17.

Observação 1.6.1.

Quando a base é o número de Euler e≈2,718281828459045, chamamos f⁢(x)=ex de função exponencial natural.

No SymPy, o número de Euler é obtido com a constante E:

>>> float(E)
2.718281828459045
