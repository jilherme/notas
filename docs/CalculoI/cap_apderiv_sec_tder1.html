5.4 Teste da primeira derivada

Na Seção 5.2, vimos que os extremos de uma função ocorrem nos extremos de seu domínio ou em um ponto crítico. Aliado a isso, o Corolário 5.3.3 nos fornece condições suficientes para classificar os pontos críticos como extremos locais.

Mais precisamente, seja c um ponto crítico de uma função contínua f e diferenciável em todos os pontos de um intervalo aberto (a,b) contendo c, exceto possivelmente no ponto c. Movendo-se no sentido positivo em x:

  • •

    se f′⁢(x) muda de negativa para positiva em c, então f possui um mínimo local em c;

  • •

    se f′⁢(x) muda de positiva para negativa em c, então f possui um máximo local em c;

  • •

    se f′ não muda de sinal em c, então c não é um extremo local de f.

Veja a Figura 5.14.

Exemplo 5.4.1.

Consideremos a função f⁢(x)=x33-2⁢x2+3⁢x+3. Como f é diferenciável em toda parte, seus pontos críticos são aqueles tais que

f′⁢(x)=0. (5.62)

Temos f′⁢(x)=x2-4⁢x+3. Segue, que os pontos críticos são

x2-4⁢x+3=0 ⇒x=4±16-1222 (5.63)
⇒x1=1,x2=3. (5.64)

Com isso, temos

Intervalo x<1 1<x<3 3<x
f′ + - +
f crescente decrescente crescente

Então, do teste da primeira derivada, concluímos que x1=1 é ponto de máximo local e que x2=3 é ponto de mínimo local.

Podemos usar o SymPy  para computarmos a derivada de f com o comando77 7 Veja a Observação 5.0.1.

fl = diff(x**3/3-2*x**2+3*x+3)

Então, podemos resolver f′⁢(x)=0 com o comando

solve(fl)

e, por fim, podemos fazer o estudo de sinal da f′ com os comandos

reduce_inequalities(fl<0)
reduce_inequalities(fl>0)

Exercícios resolvidos

ER 5.4.1.

Determine e classifique os extremos da função

f⁢(x)=x4-4⁢x3+4⁢x2. (5.65)
Solução.

Como o domínio da f é (-∞,∞) e f é diferenciável em toda parte, temos que seus extremos ocorrem em pontos críticos tais que

f′⁢(x)=0. (5.66)

Resolvendo, obtemos

4⁢x3-12⁢x2+8⁢x=0 ⇒4⁢x⁢(x2-3⁢x+2)=0 (5.67)

Logo,

4⁢x=0 ou x2-3⁢x+2=0 (5.68)
x1=0 x=3±12. (5.69)
x2=1,x3=2 (5.70)

Portanto, os ponto críticos são x1=0, x2=1 e x3=2. Fazendo o estudo de sinal da f′, temos

x<0 0<x<1 1<x<2 2<x
4⁢x - + + +
x2-3⁢x+2 + + - +
f′⁢(x) - + - +
f decrescente crescente decrescente crescente

Então, do teste da primeira derivada, concluímos que x1=0 é ponto de mínimo local, x2=-2 é ponto de máximo local e x3=-1 é ponto de mínimo local.

Podemos usar os seguintes comandos do SymPy88 8 Veja a Observação 5.0.1. para resolvermos este exercício:

# f’
fl = Lambda(x, diff(x**4 - 4*x**3 + 4*x**2,x))
# f’(x) = 0
solve(fl(x))
# fl(x) < 0
reduce_inequalities(fl(x)<0)
# fl(x) > 0
reduce_inequalities(fl(x)>0)
ER 5.4.2.

Encontre o valor máximo global de f⁢(x)=(x-1)⁢e-x.

Solução.

Como f é diferenciável em toda parte, temos que seu máximo ocorre em ponto crítico tal que

f′⁢(x)=0 ⇒(2-x)⁢e-x=0 (5.71)
⇒2-x=0 (5.72)
⇒x=2. (5.73)

Fazendo o estudo de sinal da derivada, obtemos

x<0 0<x
f’ + -
f crescente decrescente

Portanto, do teste da primeira derivada, podemos concluir que x=2 é ponto de máximo local. O favor da função neste ponto é f⁢(2)=e-2. Ainda, temos

limx→-∞⁡(x-1)⁢e-x=-∞, (5.74)
limx→∞⁡(x-1)⁢e-x=0. (5.75)

Por tudo isso, concluímos que o valor máximo global de f é f⁢(2)=e-2.

Podemos usar os seguintes comandos do SymPy99 9 Veja a Observação 5.0.1. para resolvermos este exercício:

# f(x)
f = Lambda(x, (x-1)*exp(-x))
# f’(x)
fl = Lambda(x, diff(f(x),x))
# pontos críticos
xc = solve(fl(x))
# f’(x) < 0
reduce_inequalities(fl(x)<0)
# f’(x) > 0
reduce_inequalities(fl(x)>0)
# lim f(x), x->-oo
limit(f(x),x,-oo)
# lim f(x), x->oo
limit(f(x),x,oo)
# f(2)
f(xc[0])
