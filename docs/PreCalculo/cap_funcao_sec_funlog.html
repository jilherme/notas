3.10 Funções logarítmicas

A função logarítmica y=loga⁡x, a>0 e a≠1, é a função inversa da função exponencial y=ax. Veja a Figura 3.35. O domínio da função logarítmica é (0,∞) e a imagem (-∞,∞).

Observação 3.10.1.

Quando a base é o número de Euler e≈2,718281828459045, chamamos y=loge⁡x de função logarítmica natural e denotamo-la por y=ln⁡x.

No SymPy, podemos computar loga⁡x com a função log(x,a). O ln⁡x é computado com log(x).

Observação 3.10.2.

Vejamos algumas propriedades dos logaritmos:

  • •

    loga⁡x=y⇔ay=x;

  • •

    loga⁡1=0;

  • •

    loga⁡a=1;

  • •

    loga⁡ax=x;

  • •

    alogax=x;

  • •

    loga⁡x⁢y=loga⁡x+loga⁡y;

  • •

    loga⁡xy=loga⁡x-loga⁡y;

  • •

    loga⁡xr=r⋅loga⁡x.

  • •

    loga⁡x=logb⁡xlogb⁡a

Exercícios resolvidos

ER 3.10.1.

Faça o esboço do gráfico de f⁢(x)=ln⁡(x+2)+1 e determine seu domínio.

Solução.

Para fazermos o esboço do gráfico de f⁢(x)=ln⁡(x+2)+1, podemos começar com o gráfico de f1⁢(x)=ln⁡x. Então, podemos transladá-lo 2 unidades à esquerda, de forma a obtermos f2⁢(x)=ln⁡(x+2)=f1⁢(x+2). Por fim, transladamos o gráfico de f2⁢(x) uma unidade para cima, obtendo o esboço do gráfico de f⁢(x)=ln⁡(x+2)+1=f2⁢(x)+1. Veja a Figura 3.36.

Ainda, o domínio de ln⁡x é (0,∞). Como, f⁢(x)=ln⁡(x+2)+1 é uma translação de duas unidades à esquerda e uma para cima de ln⁡x, temos que o domínio de f⁢(x) é (-2,∞).

ER 3.10.2.

Resolva a seguinte equação para x

ln⁡(x+2)+1=1. (3.146)
Solução.

Podemos calcular a solução pelos seguintes passos:

ln⁡(x+2)+1=1 (3.147)
ln⁡(x+2)=0 (3.148)
eln⁡(x+2)=e0 (3.149)
x+2=e0 (3.150)
x=1-2=-1. (3.151)

Com o SymPy, podemos computar a solução com os seguintes comandos:

⬇
1     from sympy import *
2     solve(Eq(log(x+2)+1,1),x)
