3.6 Aplicação: Equação do Calor

3.6.1 Problema Unidimensional

Consideramos o problema
ut=ux⁢x+f,(t,x)∈(0,1]×(-1,1), (3.23a)
u⁢(0,x)=sen⁡(π⁢x),x∈[-1,1], (3.23b)
u⁢(t,-1)=u⁢(t⁢,1)=0,t∈(t0,t⁢f], (3.23c)

onde f⁢(t,x)=(π2-1)⁢e-t⁢sen⁡(π⁢x) é a fonte. Este problema foi manufaturado a partir da solução

u⁢(t,x)=e-t⁢sen⁡(π⁢x). (3.24)

MLP com Diferenças Finitas

Assumimos a discretização no tempo t(k)=k⁢ht, k=0,1,2,…,nt, com passo ht=1/nt. Para a discretização no espaço, assumimos xi=-1+i⁢hx, i=0,1,2,…,nx, com passo hx=2/nx. Ainda, denotando ui(k)≈u⁢(t(k),xi), usamos as seguintes fórmulas de diferenças finitas

ut⁢(t(k),xi)≈ui(k)-ui(k-1)ht, (3.25)

para 0<k<nt, 0≤i≤nx e

ux⁢x⁢(t(k),xi)≈ui-1(k)-2⁢ui(k)+ui+1(k)hx2, (3.26)

para 0≤k≤nt e 0<i<nx.
