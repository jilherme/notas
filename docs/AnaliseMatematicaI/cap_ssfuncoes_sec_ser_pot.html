6.4 Séries de potências

Séries de potências são séries da forma

∑an⁢xn. (6.42)
Exemplo 6.4.1.

A expansão em série de Taylor da função f⁢(x)=ex em torno do ponto x=0 é

f⁢(x) =f⁢(0)0!+f′⁢(0)1!⁢x+f′′⁢(0)2!⁢x2+f′′′⁢(0)3!⁢x3+⋯ (6.43)
=1+11!⁢x+12!⁢x2+13!⁢x3+⋯ (6.44)
ex =∑n=0∞1n!⁢xn. (6.45)

Mostraremos esta afirmação no decorrer desta seção.

Lema 6.4.1.

Se a série de potências ∑an⁢xn converge absolutamente para um dado x=x0≠0, então ela é absolutamente convergente para todo |x|<|x0|. Agora, se ela diverge para um dado x=x0≠0, então ela diverge para todo |x|>|x0|.

Demonstração.

Analisemos, primeiro, o caso em que ∑an⁢xn converge absolutamente para um dado x=x0≠. Então, |an⁢x0n|→0 quando n→∞. Seja, então, M>0 tal que |an⁢x0n|<M. Com isso, temos

|an⁢xn|=|an⁢x0n|⁢|xx0|n<M⁢|xx0|n. (6.46)

Logo

∑|an⁢xn|≤M⁢∑|xx0|n. (6.47)

Esta última, é uma série geométrica e, portanto, converge sempre que |x|<|x0|. Isto mostra que ∑an⁢xn é absolutamente convergente sempre que |x|<|x0|.

Analisemos, agora, o caso em que ∑an⁢xn é divergente para um dado x=x0≠0. Neste caso, pelo que acabamos de mostrar, a série não pode convergir para nenhum x tal que |x|>|x0| pois, caso contrário, ela teria de ser convergente para x=x0. ∎

Teorema 6.4.1.

Para cada série de potência ∑an⁢xn que seja convergente em um ponto x′≠0 e diverge em um ponto x′′, existe r>0 tal que a série é absolutamente convergente se |x|<r e diverge se |x|>r.

Demonstração.

Seja r o supremo dos números |x| tal que a série de potências é convergente. Do lema anterior (Lema 6.4.1), temos |x′|≤r≤|x′′|. Então, para cada x tal que |x|<r, existe x0 tal que |x|<|x0|<r e no qual a série ∑an⁢xn é convergente. Logo, pelo lema anterior, esta série é absolutamente convergente em x. Isto mostra que a série de potências é absolutamente convergente para todo x tal que |x|<r. Por outro lado, se x é ta que |x|>r, então, também pelo lema anterior, ∑an⁢xn é divergente. ∎

Definição 6.4.1.

(Raio de convergência) Dizemos que r≤0 é o raio de convergência de uma dada série de potências ∑an⁢xn quando, esta é convergente para todo |x|<r.

Observação 6.4.1.

O Teorema 6.4.1 nos garante a existência do raio de convergência, levando em conta de que quando a série é convergente para todo x∈ℝ, dizemos que seu raio de convergência é infinito.

Teorema 6.4.2.

Se r>0 é o raio de convergência da série de potências ∑an⁢xn, então esta converge uniformemente em todo intervalo [-c,c], onde 0<c<r.

Demonstração.

Sejam c<r e x0 tal que c<x0<r. Então, ∑|an⁢x0n| é convergente e, logo, existe M tal que |an⁢x0n|<M para todo n. Assim sendo, para todo |x|<c temos

|an⁢xn|=|an⁢xn|⁢|xx0|n≤M⁢|cx0|n. (6.48)

Agora, como ∑M⁢|c/x0|n é convergente, temos do teste de Weierstrass (Teorema 6.3.4), que ∑an⁢xn é uniformemente convergente em |x|<c. ∎

Observação 6.4.2.

Do teste da raiz para séries de números, temos que para cada x tal que lim⁡sup⁡|an⁢xn|n=lim⁡sup⁡|x|⁢|an|n<1 a série ∑an⁢xn é convergente. Também, para cada x tal que lim⁡sup⁡|x|⁢|an|n>1 a série é divergente. Daí, temos que o raio de convergência de uma dada série ∑an⁢xn é

r=1lim⁡sup⁡|an|n. (6.49)
Teorema 6.4.3.

Se a série ∑n=0∞an⁢xn tem raio de convergência r>0, então

∑n=1∞n⁢an⁢xn-1, (6.50)

obtida da derivação termo a termo, também tem raio de convergência r.

Demonstração.

Seja r o raio de convergência da série ∑n=0∞an⁢xn. Primeiramente, observamos que

∑n=1∞n⁢an⁢xn-1=∑n=0∞(n+1)⁢an+1⁢xn. (6.51)

Então, da Observação 6.4.2 temos que o raio de convergência desta série é

1lim⁡sup⁡|(n+1)⁢an+1|n+1 =1lim⁡sup⁡|n⁢an|n (6.52)
=1lim⁡sup⁡nn⁢lim⁡sup⁡|an|n (6.53)
=11⋅lim⁡sup⁡|an|n=r. (6.54)

∎

Teorema 6.4.4.

Se a série ∑n=0∞an⁢xn tem raio de convergência r>0, então

∑n=0∞ann+1⁢xn+1, (6.55)

obtida da integração termo a termo, também tem raio de convergência r.

Demonstração.

A demonstração é análoga a do teorema anterior (Teorema 6.4.3). Veja o Exercício 6.4.5. ∎

Teorema 6.4.5.

(Unicidade de séries de potências) Sejam ∑an⁢xn e ∑bn⁢xn duas séries de potências com raio de convergência r. Se ∑an⁢xn=∑bn⁢xn para todo x tal que |x|<r, então an=bn para todo n.

Demonstração.

Suponhamos que f tenha as seguintes duas representações em séries de potências

f⁢(x)=∑an⁢xn=∑bn⁢xn (6.56)

para todo x tal que |x|<r. Então, pelo Teorema 6.4.3, podemos derivar termo a termo estas representações para cada x tal que |x|<r, em particular em x=0. Daí, segue o resultado desejado, veja Exercício 6.4.6. ∎

∑n=N+1∞|an⁢x0n|<∞, (6.57)

para algum x0≠0, então ∑an⁢xn é absolutamente convergente para todo |x|<|x0|.
