2.4 Limites no infinito e limites infinitos

Definição 2.4.1.

(Limites infinitos) Sejam f:D→ℝ, y=f⁢(x), e a um ponto de acumulação de D. Dizemos que o limite de f⁢(x) é +∞ quando x→a se, para todo k>0 existe δ>0 tal que para todo x∈D, 0<|x-a|<δ temos f⁢(x)>k. Analogamente, dizemos que o limite de f⁢(x) é -∞ quando x→a se, para todo k>0 existe δ>0 tal que para todo x∈D, 0<|x-a|<δ temos f⁢(x)<-k. Nestes casos escrevemos

limx→a⁡f⁢(x)=+∞  ou  limx→a⁡f⁢(x)=-∞, (2.16)

respectivamente.

Exemplo 2.4.1.

Vejamos os seguintes casos:

  1. a)

    limx→0⁡1/|x|=+∞. De fato, dado k>0 basta tomarmos δ=1/k. Com isso, 0<|x-0|<δ implica |x|<1/k e, portanto, 1/|x|>k, i.e. |1/|x|-0|>k.

  2. b)

    Seja f:(-∞,0)→ℝ, y=f⁢(x):=1/x. Neste caso, limx→0⁡f⁢(x)=-∞. Deixamos a verificação para o leitor.

Definição 2.4.2.

(Limite no infinito) Seja f:D→ℝ, y=f⁢(x). Quando D é ilimitado superiormente dizemos que f⁢(x) tende a L quando x→+∞ se, para todo ε>0 existe k>0 tal que x>k implica |f⁢(x)-L|<ε. Analogamente, quando D é ilimitado inferiormente dizemos que f⁢(x) tende a L quando x→-∞ se, para todo ε>0 existe k>0 tal que x<-k implica |f⁢(x)-L|<ε. Nestes casos escrevemos

limx→+∞⁡f⁢(x)=L  ou  limx→-∞⁡f⁢(x)=L, (2.17)

respectivamente.

Exemplo 2.4.2.

Vejamos os seguintes casos:

  1. a)

    limx→∞⁡1/x=0. De fato, dado ε>0 escolhemos δ=1/ε. Com isso, x>δ implica 0<1/x<1/δ=ε e, portanto, |1/x-0|<ε.

  2. b)

    limx→-∞⁡1/x=0. Caso análogo ao anterior, verifique!

Observação 2.4.1.

Observe que definições análogas às 2.3.1, 2.4.1 e 2.4.2 se aplicam para os casos:

limx→a+⁣/-f(x)=±/∓∞  ou  limx→±∞f(x)=L. (2.18)

Também, consideramos definições análogas para os casos:

limx→a+⁣/-⁡f⁢(x)=L±⁣/∓  ou  limx⁣→⁣+⁣/-∞⁡f⁢(x)=L±⁣/∓. (2.19)
Teorema 2.4.1.

Toda função monótona e limitada superiormente (inferiormente), cujo domínio contenha [c,+∞) ((∞,c]), possui limite quando x→+∞ (x→-∞).

Demonstração.

Consideremos o caso de f:[c,+∞)→ℝ, y=f⁢(x), função não decrescente e limitada superiormente. Seja, então L o supremo do conjunto imagem de f. Mostraremos que f⁢(x)→L quando x→+∞. De fato, dado ε>0 existe k>0 tal que L-ε<f⁢(k)≤L. Agora, como f é não decrescente, para todo x>k temos L-ε<f⁢(k)<f⁢(x)≤k e, portanto, |f⁢(x)-L|<ε. Isto demostra o caso considerado e deixamos para o leitor a verificação dos demais (veja, também, Exercício 2.4.1). ∎

Exercícios

E 2.4.1.

) Demonstre o Teorema 2.4.1 para o caso de uma função decrescente e limitada inferiormente.

Resp.

Análogo ao caso demonstrado no Teorema 2.4.1.
