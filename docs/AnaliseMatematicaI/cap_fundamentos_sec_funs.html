1.1 Funções

1.1.1 Definição de função

Definição 1.1.1.

(Função) Uma função f:D→Y é uma relação que associa cada elemento de um dado conjunto D com um único elemento de um dado conjunto Y. O conjunto D é chamado de domínio da função e o conjunto Y é chamado de contradomínio da função.

Comumente, uma dada função f:D→Y é acompanhada de sua lei de correspondência, a qual muitas vezes é denotada por y=f⁢(x). Neste caso, temos que a função f associa x∈D ao elemento y∈Y. Neste contexto, x é chamada de variável independente e y de variável dependente. Ainda, muitas vezes uma função é descrita apenas por sua lei de correspondência e, neste caso, os conjuntos domínio e imagem são inferidos no contexto em questão.

Observação 1.1.1.

Neste livro, quando não especificado ao contrário, assumiremos que o domínio e o contradomínio das funções consideradas são subconjuntos dos números reais,

Exemplo 1.1.1.

Vejamos os seguintes casos:

  1. a)

    A relação f:{1,2,3}→ℝ, y=f⁢(x):=x2+1, define uma função.

  2. b)

    A relação g:D={0,1,2,3,4}→ℤ, x2+y2=9 com x∈D e y∈Y, não é uma função. Com efeito, 0∈D e relaciona-se com 3∈Y e -3∈Y no seu contradomímio.

  3. c)

    Da equação y=x pode-se inferir a função h:x∈D→y∈ℝ, onde o domínio D é conjunto dos reais não negativos.

Definição 1.1.2.

(Imagem de uma função) A imagem If de uma dada função f:D→Y é o conjunto de todos os elementos de Y que se relacionam com algum elemento de D, i.e.:

If:={y∈Y;∃x∈D tal que y=f(x)}. (1.1)
Exemplo 1.1.2.

Vejamos os seguintes casos:

  1. a)

    A função f:{1,2,3}→ℝ, y=f⁢(x):=x2+1, tem imagem If={1,4,9}.

  2. b)

    A imagem da função f:{0}∪ℕ→ℝ, y=2⁢x+1, é conjunto dos números ímpares.

  3. c)

    A imagem da função sen:ℝ→ℝ, y=sen⁡x, é Isen=[-1,1].

Observação 1.1.2.

Dada uma função f:D→Y e um conjunto A⊂D, definimos a imagem de A pela função f por

f(A):={y∈Y;∃x∈A tal que y=f(x)}. (1.2)

Por exemplo, dada a função f:ℝ→ℝ, y=x, temos

f⁢({0,1,4,9})={0,1,2,3}. (1.3)
Definição 1.1.3.

(Gráfico) O gráfico de uma função f:D→Y, y=f⁢(x), é o conjunto de todos os pares ordenados (x,y) tal que x∈D e y=f⁢(x), i.e.

Gf:={(x,y)∈D×Y;y=f(x)}. (1.4)
Exemplo 1.1.3.

O gráfico da função f:{1,2,3}→ℝ, y=f⁢(x):=x2+1, é

Gf={(1,2),(2,5),(3,10)}. (1.5)

1.1.2 Classificações elementares

Definição 1.1.4.

(Função limitada) Seja dada uma função f:D→ℝ, y=f⁢(x). Dizemos que f é uma função limitada inferiormente (ou limitada à esquerda) quando existe m∈ℝ tal que m≤f⁢(x) para todo x∈D. Analogamente, dizemos que f é uma função limitada superiormente (ou limitada à direta) quando existe M∈ℝ tal que f⁢(x)≥M para todo x∈D. Ainda, f é dita ser limitada quando é limitada inferiormente e superiormente.

Exemplo 1.1.4.

Vejamos os seguintes casos:

  1. a)

    A função f:ℝ→ℝ, y=x2+1, é limitada inferiormente. De fato, para cada x∈ℝ temos x2≥0 e, portanto, y=x2+1≥1.

  2. b)

    A função seno é uma função limitada. Isto segue imediatamente da definição da função seno no círculo unitário (círculo trigonométrico).

Definição 1.1.5.

Restrição/extensão de uma função Uma função g:A→Y, y=g⁢(x), é dita ser uma restrição da dada função f:D→Y quando A⊂D e g⁢(x)=f⁢(x) para todo x∈A. Analogamente, f é uma extensão da função g.

Exemplo 1.1.5.

A função f:ℝ→ℝ, y=x+1, é uma extensão da função g:ℝ∖{1}→ℝ, y=x2-1x-1.

Definição 1.1.6.

(Função injetiva) Uma função f:D→Y, y=f⁢(x), é dita ser injetiva (injetora ou inversível) quando para todo x1,x2∈D com x1≠x2 temos f⁢(x1)≠f⁢(x2).

Observação 1.1.3.

Uma função f:D→ℝ, y=f⁢(x), é injetiva se, e somente se, para todo x1,x2∈D tal que f⁢(x1)=f⁢(x2) temos x1=x2.

Exemplo 1.1.6.

Vejamos os seguintes casos:

  1. a)

    A função f⁢(x)=x2 não é injetiva, pois tomando x1=-1 e x2=1 temos x1≠x2, mas f⁢(x1)=f⁢(x2).

  2. b)

    A função f⁢(x)=x+1 é injetiva. De fato, dados x1,x2∈𝔻 tal que f⁢(x1)=f⁢(x2), então x1=x2. Agora, tomando o quadrado dos dois lados, temos x1=x2.

Definição 1.1.7.

(Função sobrejetiva) Uma função f:D→Y, y=f⁢(x), é sobrejetiva quando f⁢(D)=Y (ou, equivalentemente, If=Y).

Exemplo 1.1.7.

A função f:(0,∞)→ℝ, f⁢(x)=ln⁡(x), é sobrejetiva. De fato, dado qualquer y∈ℝ basta escolhermos x=ey para termos f⁢(x)=y.

Observação 1.1.4.

Uma função injetiva e sobrejetiva é dita ser bijetiva.

Definição 1.1.8.

(Função inversa) Dada uma função invertível (i.e. injetora) f:D→Y, y=f⁢(x), definimos sua inversa por f-1:f⁢(D)→D que associa cada elemento y∈f⁢(D) com x∈D tal que f⁢(x)=y.

Exemplo 1.1.8.

Vejamos os seguintes casos:

  1. a)

    A inversa da função f:(0,∞)→ℝ, y=ln⁡(x), é a função f-1:ℝ→(0,∞), y=ex.

  2. b)

    A inversa da função f:[-1,∞]→[0,∞), y=x+1, é a função f-1:[0,∞)→[-1,∞], y=x2-1. De fato, f é sobrejetiva e dado x∈[-1,∞] temos f⁢(x)=y=x+1 e, então y2=x+1, logo x=y2-1.

Definição 1.1.9.

(Função monótona) Seja dada uma função f:D→Y. Dizemos que f é crescente quando para todo x1,x2∈D com x1<x2, temos f⁢(x1)<f⁢(x2). Agora, quando para todo x1,x2∈D com x1<x2 temos f⁢(x1)≤f⁢(x2), dizemos que f é uma função não-decrescente. Analogamente, quando para todo x1,x2∈D com x1<x2 temos f⁢(x1)>f⁢(x2) dizemos que f é uma função decrescente. Por fim, quando para todo x1,x2∈D com x1<x2 temos f⁢(x1)≥f⁢(x2) dizemos que f é uma função não-crescente.

Exemplo 1.1.9.

Vejamos os seguintes casos:

  1. a)

    f:ℝ→ℝ, y=x3, é uma função crescente.

  2. b)

    f:ℝ→ℝ, y=e-x é uma função decrescente.

Definição 1.1.10.

(Paridade de uma função) Uma função f:D→Y, y=f⁢(x), é dita ser par quando para todo x∈D, temos f⁢(x)=f⁢(-x). Agora, quando para todo x∈D, temos f⁢(x)=-f⁢(-x), então dizemos se tratar de uma função ímpar.

Exemplo 1.1.10.

Vejamos os seguintes casos:

  1. a)

    A função f:ℝ→ℝ, y=|x|, é uma função par.

  2. b)

    A função f:ℝ→ℝ, y=x3, é uma função ímpar.

1.1.3 Operações elementares

Operações elementares envolvendo funções são comumente definidas tomando o cuidado de restringir o domínio das funções operadas para um conjunto apropriado. Por exemplo, dadas as funções f:A→ℝ, y=f⁢(x), e g:B→ℝ, y=g⁢(x), definimos a função soma de f com g por (f+g):A∩B→ℝ, (f+g)⁢(x):=f⁢(x)+g⁢(x). Agora, para estas mesmas função, definimos a função quociente de f com g por (f/g):A∩B∖{0}→ℝ, (f/g)⁢(x):=f⁢(x)/g⁢(x).

Exemplo 1.1.11.

A função f:[0,∞]→ℝ, y=x-|x|, é a subtração da função f1:[0,∞]→ℝ, y=x, com a função f2:ℝ→ℝ, y=|x|, i.e. f⁢(x)=(f1-f2)⁢(x):=f1⁢(x)-f2⁢(x).

Definição 1.1.11.

(Composição de funções) Sejam dadas as funções f:Df→Yf, y=f⁢(x), e g:Dg→Yg, y=g⁢(x), com Ig⊂Df. Definição a função composta de f com g por (f∘g):Dg→Yf com (f∘g)⁢(x)=f⁢(g⁢(x)).

Exemplo 1.1.12.

A função f:[0,∞]→ℝ, y=x2+1, é a composição da função f1:[0,∞]→ℝ, y=x, com a função f2:ℝ→ℝ, y=x2+1.

Exercícios

E 1.1.1.

Sejam f:D→Y, y=f⁢(x), e A,B⊂D. Mostre que f⁢(A∪B)=f⁢(A)∪f⁢(B).

E 1.1.2.

Construa uma função crescente, limitada superiormente e com domínio igual ao conjunto dos números reais.

E 1.1.3.

Mostre que f:[1,∞)→ℝ, y=x3-1, é injetora e construa sua inversa.

E 1.1.4.

Mostre que se f:D→Y é injetora, então f não é par.

E 1.1.5.

Mostre que uma dada função f:ℝ→ℝ, y=f⁢(x), é limitada quando existe c∈ℝ tal que |f⁢(x)|<c, ∀x∈ℝ.
