2.1 Noções de topologia

Definição 2.1.1.

(Ponto interior) Diz-se que x é um ponto interior de um dado conjunto C quando existe um intervalo (a,b) que contém x e está contido em C, i.e. x∈(a,b)⊂C. O conjunto de todos os pontos interiores de C é chamado de seu interior.

Exemplo 2.1.1.
  1. a)

    Todo elemento de um intervalo aberto (a,b) é ponto interior deste.

  2. b)

    O interior de um dado intervalo fechado [a,b] é o intervalo aberto (a,b).

Definição 2.1.2.

(Conjunto aberto) Diz se que C é conjunto aberto quando todos seus elementos são pontos interiores.

Exemplo 2.1.2.

Vejamos os seguintes casos:

  1. a)

    O intervalo (a,b):={x∈ℝ;a<x<b} é um conjunto aberto. De fato, dado x∈(a,b) podemos tomar 0<ε<min⁡{x-a,b-x} de forma que x∈(x-ε,x+ε)⊂(a,b).

  2. b)

    O intervalo (a,b] não é aberto, pois b∈(a,b] não é ponto interior.

  3. c)

    O conjunto vazio ∅ é um conjunto aberto. Com efeito, se o conjunto ∅ não é aberto, então existe um elemento x∈∅ que não é ponto interior de ∅, o que é um absurdo pois ∅ não contém elementos por definição.

  4. d)

    O conjunto dos números racionais ℚ não é aberto.

Definição 2.1.3.

(Vizinhança) Uma vizinhança de um dado ponto x é qualquer conjunto V que contenha x como ponto interior. Também, a vizinhança simétrica de um ponto x∈ℝ é todo intervalo Vε⁢(x):=(x-ε,x+ε) com ε>0. Mais estrito, a vizinhança perfurada de x∈ℝ é uma vizinhança de x que não contém x. Aproveitamos para fixar a notação:

Vε′(x):=Vε(x)∖{x}={y∈ℝ; 0<|x-y|<ε}.
Exemplo 2.1.3.

Podemos reescrever o Exemplo 2.1.2 da seguinte forma. Um intervalo (a,b) é um conjunto aberto, pois para cada x∈(a,b) podemos escolher 0<ε<min⁡{x-a,b-x} tal que Vε⁢(x)⊂(a,b).

Definição 2.1.4.

(Ponto de acumulação) Um ponto x é chamado de ponto de acumulação de um dado conjunto C quando toda vizinhança de x contém infinitos pontos de C.

Exemplo 2.1.4.

Vejamos os seguintes casos:

  1. a)

    O número a é ponto de acumulação do intervalo (a,b] não degenerado. De fato, dado ε>0, temos (a,a+ε)⊂Vε⁢(a) e (a,a+ε)∩(a,b] é um conjunto infinito.

  2. b)

    Zero é o único ponto de acumulação do conjunto {1,1/2,1/3,…,1/n,…}.

Definição 2.1.5.

(Ponto isolado) Diz que x é ponto isolado de um dado conjunto C quando x∈C não é ponto de acumulação de C. Diz-se que um conjunto é discreto quando todos seus elementos são pontos discretos.

Exemplo 2.1.5.

Vejamos os seguintes casos:

  1. a)

    O conjunto dos números naturais ℕ é discreto.

  2. b)

    O conjunto dos números racionais ℚ não é discreto.

  3. c)

    O conjunto {1,1/2,1/3,…,1/n,…} é discreto.

Definição 2.1.6.

(Ponto aderente) Dizemos que x é ponto aderente de um dado conjunto C quando toda vizinhança de x contém algum ponto de C. O conjunto de todos os pontos aderentes de C é chamado de fecho (ou, conjunto de aderência) de C, o qual denotamos por C¯.

Observação 2.1.1.

Observe que todo ponto de um conjunto é aderente ao mesmo, bem como, todos os seus pontos de acumulação.

Exemplo 2.1.6.

Vejamos os seguintes casos:

  1. a)

    O fecho de (a,b] é o intervalo fechado [a,b].

  2. b)

    O conjunto dos números reais ℝ é o fecho do conjunto dos números racionais ℚ, i.e. Q¯=ℝ.

Definição 2.1.7.

Conjunto fechado Dizemos que um conjunto C é fechado quando é igual ao seu fecho, i.e. C=C¯.

Exemplo 2.1.7.

Vejamos os seguintes casos:

  1. a)

    O intervalo [a,b] é um conjunto fechado.

  2. b)

    O conjunto vazio ∅ é fechado. Por quê?

  3. c)

    O conjunto dos números reais ℝ é fechado.

  4. d)

    O conjunto dos números racionais ℚ não é fechado.

Definição 2.1.8.

(Conjunto denso) Dizemos que um conjunto A é denso no conjunto B, quando todo ponto aderente de A¯⊂B.

Exemplo 2.1.8.

O conjunto dos números racionais ℚ é denso no conjunto dos números reais ℝ.

Exercícios

E 2.1.1.

Seja dado um conjunto C. Mostre que x é ponto de acumulação de C se, e somente se, toda vizinhança de x contém pelo menos um elemento de C diferente de x.

Resp.

Basta considerar sucessivas vizinhanças V1/n⁢(x) com n∈ℝ.

E 2.1.2.

Seja dado um conjunto C. Mostre que x é ponto isolado de C se, e somente se, existe uma vizinhança de x tal que (V⁢(x)∖{x})∩C=∅.

Resp.

A implicação segue imediatamente por negação.
