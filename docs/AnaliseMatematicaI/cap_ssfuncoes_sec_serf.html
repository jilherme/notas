6.3 Séries de funções

Definição 6.3.1.

(Séries de funções) Sejam dadas funções fi:D→ℝ, i=1,2,3,…. A série das funções fi é a sequência das somas parciais

∑i=1nfi⁢(x):=f1⁢(x)+f2⁢(x)+f3⁢(x)+⋯+fn⁢(x). (6.26)

Comumente, denotamos uma tal série por

∑i=1∞fi⁢(x). (6.27)

Tendo em vista de uma série de funções ∑ifi⁢(x) é uma sequência de funções, os conceitos de convergência de séries são aplicações diretas das definições de convergência de sequências de funções. Ou seja, dizemos que uma série de funções é pontualmente convergente para a função f:D→ℝ quando, para cada ε>0 e para cada x∈D, existe N tal que

n>N⇒|∑i=1nfi⁢(x)-f⁢(x)|=|∑i=n+1∞fi⁢(x)|<ε. (6.28)

Ainda, dizemos uma série de funções é uniformemente convergente quando, para cada ε>0, existe N tal que

x∈D,n>N⇒|∑i=n+1∞fi⁢(x)|<ε. (6.29)

Assim como as definições de séries de funções, os seguintes teoremas são consequências diretas dos relacionados a sequências de funções.

Teorema 6.3.1.

(Critério de Cauchy) Uma série de funções ∑fi⁢(x), fi:D→ℝ, é uniformemente convergente se, e somente se, para qualquer ε>0, existe N tal que

x∈D,p∈ℕ,n>N⇒|∑i=n+1n+pfn+i⁢(x)|<ε. (6.30)
Demonstração.

Do critério de convergência de Cauchy para sequências de funções, temos que ∑fi⁢(x) é uniformemente convergente se, e somente se, para qualquer ε>0, existe N tal que

x∈D,n,m>N⇒|∑i=1nfi⁢(x)-∑i=1mfi⁢(x)|<ε. (6.31)

Agora, assumindo m>n s.p.g. e denotando p=m-n temos

|∑i=1nfi⁢(x)-∑i=1mfi⁢(x)| =|∑i=1mfi⁢(x)-∑i=1nfi⁢(x)| (6.32)
=|∑i=m+1nfi⁢(x)| (6.33)
=|∑i=n+1mfi⁢(x)| (6.34)
=|∑i=n+1n+pfi⁢(x)|. (6.35)

∎

Teorema 6.3.2.

Seja ∑fi⁢(x) uma série de funções fi:D→R contínua em D. Se ∑fi⁢(x) converge uniformemente para f:D→R, então f é contínua em D.

Demonstração.

Sejam

Sn⁢(x)=∑i=1nfi⁢(x) (6.36)

as somas parciais da série ∑fi⁢(x). Como a soma finita de funções contínuas em um conjunto D é uma função contínua no mesmo, temos que Sn é uma função contínua. Logo, como Sn→f uniformemente, temos do Teorema 6.2.1 que f é contínua em D. ∎

Teorema 6.3.3.

Seja ∑fi⁢(x) uma série de funções fi:[a,b]→R contínua em [a,b]. Se ∑fi⁢(x) converge uniformemente para f:D→R, então

∑i=1∞∫abfi⁢(x)⁢𝑑x=∫ab∑i=1∞fi⁢(x)⁢d⁢x. (6.37)

Teorema 6.3.4.

(Teste de Weierstrass) Seja fn:D→ℝ uma sequência de funções. Se |fn⁢(x)|≤Mn, ∀x∈D, com ∑Mn convergente, então ∑fn⁢(x) converge absoluta e uniformemente.

Demonstração.

Seja ε>0. Como ∑Mn é convergente, existe N tal que

n≥N⇒|∑i=n∞Mi|≤ε. (6.38)

Mas, então, temos

x∈D,n≥N⇒|∑i=n∞fn⁢(x)| ≤∑i=n∞|fn⁢(x)| (6.39)
≤∑i=n∞Mi<ε. (6.40)

Isto mostra que ∑fn⁢(x) é absoluta e uniformemente convergente. ∎

Exemplo 6.3.1.

A série ∑sen⁡(n⁢x)/n! é absoluta e uniformemente convergente. De fato, como |sen⁡(n⁢x)|≤1 para todo x∈ℝ, temos que |sen⁡(n⁢x)/n!|≤1/n!. Ainda, como ∑1/n! é convergente, segue do Teste de Weierstrass 6.3.4, que ∑sen⁡(n⁢x)/n! é absoluta e uniformemente convergente.
