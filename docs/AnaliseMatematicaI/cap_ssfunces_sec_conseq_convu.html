6.2 Algumas consequências da convergência uniforme

Teorema 6.2.1.

Seja fn:D→R uma sequência de funções contínuas. Se fn converge uniformemente para uma função f:D→R, então f é contínua.

Demonstração.

Primeiramente, observemos que para quaisquer x,a∈D e n∈ℕ, temos

|f⁢(x)-f⁢(a)| =|f⁢(x)-fn⁢(x)+fn⁢(x)-fn⁢(a)+fn⁢(a)-f⁢(a)| (6.14)
≤|f⁢(x)-fn⁢(x)|+|fn⁢(x)-fn⁢(a)|+|fn⁢(a)-f⁢(a)|. (6.15)

Seja, então, ε>0. Como fn converge uniformemente para f, existe N∈ℕ tal que, para todo n>N temos

|f⁢(x)-fn⁢(x)|<ε3  e  |fn⁢(a)-f⁢(a)|<ε3. (6.16)

Fixemos um n>N. Como fn é contínua, para cada a∈D, existe δ>0 tal que

x∈D,|x-a|<δ⇒|fn⁢(x)-fn⁢(a)|<ε3. (6.17)

Portanto, usando a desigualdade em (6.14), vemos que para cada a∈D, existe δ>0 tal que

x∈D,|x-a|<ε⇒|f⁢(x)-f⁢(a)|<ε. (6.18)

Isto mostra a continuidade de f. ∎

Teorema 6.2.2.

Seja fn:[a,b]→R uma sequência de funções contínuas. Se fn converge uniformemente para uma função f:[a,b]→R, então

lim⁡∫abfn⁢(x)⁢𝑑x=∫ab[lim⁡fn⁢(x)]⁢𝑑x=∫abf⁢(x)⁢𝑑x. (6.19)
Demonstração.

Seja ε>0. Como fn converge uniformemente para f, temos que existe N∈ℕ tal que

x∈[a,b],n>N⇒|fn⁢(x)-f⁢(x)|<ε(b-a). (6.20)

Além disso, do teorema anterior (Teorema 6.2.1), temos que f é contínua em [a,b] e, portanto, assim como fn, é integrável neste intervalo (veja Teorema 5.2.1). Por tudo isso, temos

n>N⇒|∫abfn⁢(x)⁢𝑑x-∫abf⁢(x)⁢𝑑x| =|∫ab|⁢fn⁢(x)-f⁢(x)⁢|d⁢x| (6.21)
≤ε(b-a)⁢(b-a)=ε. (6.22)

O que concluí a demonstração. ∎

Teorema 6.2.3.

Seja fn:[a,b]→R uma sequência de funções com derivadas contínuas em [a,b], tal que fn′ converge uniformemente para uma função g:[a,b]→R. Suponhamos, ainda, para algum ponto c∈[a,b], fn⁢(c) é uma sequência convergente. Então, fn converge uniformemente para uma função f para a qual f′=g, i.e.

dd⁢x⁢lim⁡fn⁢(x)=lim⁡dd⁢x⁢fn⁢(x). (6.23)
Demonstração.

Do teorema fundamental do cálculo (Teorema 5.3.2), para todo x∈[a,b] temos

fn⁢(x)=fn⁢(c)+∫cxfn′⁢(t)⁢𝑑t. (6.24)

Como, por hipótese, fn⁢(c) é convergente e fn′ é uniformemente convergente para a função g, temos, do teorema anterior (Teorema 6.2.2), que tomando o limite de n→∞ nesta última equação, obtemos

f⁢(x)=f⁢(c)+∫cxg⁢(t)⁢𝑑t. (6.25)

Observemos que f′=g. Fica de exercício, mostrar que fn converge uniformemente para f, o que completa a demonstração. ∎

Exercícios

E 6.2.1.

Complete a demonstração do Teorema 6.2.3.
