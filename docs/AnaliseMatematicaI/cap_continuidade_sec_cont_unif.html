3.3 Continuidade uniforme

Definição 3.3.1.

(Continuidade uniforme) Uma função f:D→ℝ, y=f⁢(x), é dita ser uniformemente contínua se, dado qualquer ε>0 existe δ>0 tal que

x,y∈D,|x-y|<δ⇒|f⁢(x)-f⁢(y)|<ε. (3.7)
Exemplo 3.3.1.

Vejamos os seguintes casos:

  1. a)

    f⁢(x)=x é uniformemente contínua. De fato, consideremos x,y>0 e |x-y|<δ para δ>0 arbitrário. Então, se y<δ temos x<y+δ<2⁢δ e

    |x-y|≤x+y<2⁢δ+δ<3⁢δ. (3.8)

    Agora, se y≥δ, então

    |x-y|=|x-y|x+y<δy<δδ=δ. (3.9)

    Logo, em qualquer um dos casos |x-y|<3⁢δ. Por tanto, dado ε>0, podemos escolher δ=ε2/9 de forma que

    x,y>0,|x-y|<δ⇒|x-y|<ϵ, (3.10)

    o que conclui o resultado.

  2. b)

    A função f⁢(x)=1/x não é uniformemente contínua. De fato, basta observar que, para qualquer escolha de δ>0, temos

    |1x-1x+δ|=|δx2+δ⁢x|→+∞ com x→0. (3.11)
Teorema 3.3.1.

(de Heine) Se f:[a,b]→ℝ é contínua em [a,b]=:I, então f é uniformemente contínua.

Demonstração.

Suponhamos, por contradição, que f não é uniformemente contínua. Então, para algum ϵ>0 existem xn,yn∈I tal que

|xn-yn|<1n e |f⁢(xn)-f⁢(yn)|>ϵ, (3.12)

para todo n∈ℕ. Agora, como (xn)n é uma sequência limitada, pelo teorema de Bolzano-Weierstrass ela possui uma subsequência convergente. Seja, então, (xn′)n′ uma tal subsequência e c o seu limite. Como xn′∈[a,b] para todo n′, temos c∈[a,b]. Além disso, como |xn′-yn′|→0, temos yn′→c. Também, pela continuidade de f, temos f⁢(xn′)→f⁢(c) e f⁢(yn′)→f⁢(c). Logo, |f⁢(xn′)-f⁢(yn′)|→0, o que é um absurdo. ∎

Exercícios

E 3.3.1.

Mostre que se f é uniformemente contínua em (a,b), então existem os limites limx→a+⁡f⁢(x) e limx→b-⁡f⁢(x).

Resp.

Dica: 1) mostre que toda sequência xn∈(a,b) com xn→a é tal que f⁢(xn) é converge. Seja L o limite desta sequência; 2) mostre, então, que qualquer outra sequência yn∈(a,b) com yn→a é tal que f⁢(yn)→L.
