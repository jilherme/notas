4.5 Métodos de passo múltiplo

Necessita revisão!

Dado um problema de valor inicial

y′⁢(t) =f⁢(t,y⁢(t)),t>t0, (4.142)
y⁢(t0) =y0. (4.143)

temos

y⁢(t)=y⁢(t0)+∫t0tf⁢(s,y⁢(s))⁢𝑑s. (4.144)

De forma mais geral, consideramos uma partição uniforme no tempo {t0=t(1)<t(2)<⋯<t(i)<⋯<t(n)=tf}, onde tf é um determinado tempo para o qual queremos computar uma aproximação para y⁢(tf). Também, denotamos o passo no tempo por h=(tf-t0)/n. Com isso, a solução y⁢(t) satisfaz

y⁢(t(i+k))=y⁢(t(i-j))+∫t(i-j)t(i+k)f⁢(s,y⁢(s))⁢𝑑s. (4.145)

A ideia é, então, aproximar a integral acima por uma quadratura numérica.

Seguindo as regras de Newton-Cotes (veja, Cap. 3 Seç. 3.1), escolhemos os nodos da quadratura como xl=t(i-l+1), l=1,2,…,m, e, então

∫t(i-j)ti+kf⁢(x,y⁢(x))⁢𝑑x≈∑l=1mf⁢(xl,y⁢(xl))⁢wl, (4.146)

e

wl=∫t(i-j)t(i+k)∏p≠lp=1mx-xpxl-xp⁢d⁢x. (4.147)

Agora, fazendo a mudança de variável u=(x-t(i))/h, obtemos

wl=h⁢∫-jk∏p≠lp=1mu+p-1-l+p⁢d⁢u (4.148)

Assim sendo, temos o seguinte esquema numérico

y(i+k)=y(i-j)+h⁢∑l=1mcl⁢f⁢(t(i-l+1),y(i-l+1)), (4.149)

onde

cl=∫-jk∏p≠lp=1ms+p-1-l+p⁢d⁢s. (4.150)

Diferentes escolhas de j, k e m não fornecem diferentes métodos. Observamos, ainda, que a ordem de um tal método de passo múltiplo é determinada pela ordem de truncamento da quadratura numérica usada (veja, por exemplo, [Burden2015a, Cap. 5, Seç. 5.6]).

4.5.1 Métodos de Adams-Bashforth

Necessita revisão!

Métodos de Adams-Bashforth são métodos de passo múltiplo obtidos ao escolhermos j=0 e k=1 no esquema numérico (4.149). Com isso, ao escolhermos m obtemos um método de ordem O⁢(hm) [Burden2015a, Cap. 5, Seç. 5.6].

Método de Adams-Bashforth de ordem 2

Necessita revisão!

Tomando m=2 em (4.150), temos

c1=∫01s+1⁢d⁢s=32 (4.151)

e

c2=∫01-s⁢d⁢s=-12. (4.152)

Então, de (4.149) temos a iteração do método de Adams-Bashforth de 2 passos:

y(1) =y0, (4.153)
y(i+1) =y(i)+h2⁢[3⁢f⁢(t(i),y(i))-f⁢(t(i-1),y(i-1))], (4.154)

com t(i)=t0+(i-1)⁢h.

Exemplo 4.5.1.

Consideremos o seguinte problema de valor inicial

y′-y =sen⁡(t),t>0 (4.155)
y⁢(0) =12. (4.156)

Na Tabela 4.6, temos as aproximações y~⁢(1) de y⁢(1) computadas pelo método de Adams-Bashforth de 2 passos. Como este método é de ordem 2, escolhemos inicializá-lo pelo método do ponto médio, de forma a mantermos a consistência.

Método de Adams-Bashforth de ordem 3

Necessita revisão!

Tomando m=3 em (4.150) obtemos, de (4.149), a iteração do método de Adams-Bashforth de 3 passos:

y(1) =y0, (4.157)
y(i+1) =y(i)+h12[23f(t(i),y(i))
-16f(t(i-1),y(i-1))+5f(t(i-2),y(i-2))], (4.158)

com t(i)=t0+(i-1)⁢h.

Exemplo 4.5.2.

Consideremos o seguinte problema de valor inicial

y′-y =sen⁡(t),t>0 (4.159)
y⁢(0) =12. (4.160)

Na Tabela 4.7, temos as aproximações y~⁢(1) de y⁢(1) computadas pelo método de Adams-Bashforth de 3 passos. Como este método é de ordem 3, escolhemos inicializá-lo pelo método de Runge-Kutta de ordem 4, de forma a garantirmos a consistência.

Método de Adams-Bashforth de ordem 4

Necessita revisão!

Tomando m=4 em (4.150) obtemos, de (4.149), a iteração do método de Adams-Bashforth de 4 passos:

y(1) =y0, (4.161)
y(i+1) =y(i)+h24[55f(t(i),y(i))
-59⁢f⁢(t(i-1),y(i-1))+37⁢f⁢(t(i-2),y(i-2))
-9f(t(i-3),y(i-3))], (4.162)

com t(i)=t0+(i-1)⁢h.

Exemplo 4.5.3.

Consideremos o seguinte problema de valor inicial

y′-y =sen⁡(t),t>0 (4.163)
y⁢(0) =12. (4.164)

Na Tabela 4.8, temos as aproximações y~⁢(1) de y⁢(1) computadas pelo método de Adams-Bashforth de 4 passos. Como este método é de ordem 3, escolhemos inicializá-lo pelo método de Runge-Kutta de ordem 4, de forma a mantermos a consistência.

4.5.2 Exercícios

Necessita revisão!

Exercício 4.5.1.

Considere o seguinte problema de valor inicial

y′ +e-y2+1=2,t>1, (4.165)
y⁢(1) =-1. (4.166)

Inicializando pelo método de Euler, use os seguintes métodos de passo múltiplo com h=0,1 para computar o valor aproximado de y⁢(2):

  1. a)

    método de Adams-Bashforth de ordem 2.

  2. b)

    método de Adams-Bashforth de ordem 3.

  3. c)

    método de Adams-Bashforth de ordem 4.

Resp.

a) -6,00696⁢E-1; b) -5,96694⁢E-1; c) -5,96161⁢E-1
