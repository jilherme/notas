4.4 Método de Euler Implícito

Seja o Problema de Valor Inicial (PVI)
y′=f⁢(t,y),t0<t≤tf, (4.123a)
y⁢(t0)=y0, (4.123b)

com dados t0,tf∈ℝ e y0∈ℝ, sendo a função y=y⁢(t) a incógnita.

Consideramos a discretização no tempo t(k):=t0+h⁢k, k=0,1,2,…,n, com passo h:=(tf-t0)/n. De (4.123) e do Teorema Fundamental do Cálculo temos

y⁢(t(k+1))=y⁢(t(k))+∫t(k)t(k+1)f⁢(t,y)⁢𝑑t. (4.124)

A integral pode ser aproximada por

∫t(k)t(k+1)f⁢(t,y)⁢𝑑t≈h⁢f⁢(t(k+1),y(k+1)) (4.125)

donde obtemos

y⁢(t(k+1))≈y⁢(t(k))+h⁢f⁢(t(k+1),y(k+1)). (4.126)

Isto nos motiva a iteração do Método de Euler Implícito88 8 Leonhard Paul Euler, 1707-1783, matemático e físico suíço. Fonte: Wikipédia.

y(0)=y0, (4.127)
y(k+1)=y(k)+h⁢f⁢(t(k+1),y(k+1)),

sendo y(k)≈y⁢(t(k)), k=0,1,2,…,n.

Exemplo 4.4.1.

Consideremos o seguinte PVI
y′-y=sen⁡(t),t>0 (4.128a)
y⁢(0)=12. (4.128b)

Na Tabela 4.6, temos as aproximações y~⁢(1)≈y⁢(1) computadas pelo Método de Euler Implícito com diferentes passos h.

4.4.1 Análise Numérica

O Método de Euler Implícito é O⁢(h). De fato, tomando o polinômio de Taylor

y⁢(t)=y⁢(t+h)-h⁢f⁢(t+h,y⁢(t+h))+O⁢(h2), (4.129)

temos

τ⁢(t,y;h) :=Δ⁢(t,y;h)-Φ⁢(t,y;h) (4.130)
=y⁢(t+h)-y⁢(t)h⏟:=Δ-f⁢(t+h,y⁢(t+h);h)⏟:=Φ (4.131)
=O⁢(h). (4.132)

Estabilidade

Um método de dito ser estável quando pequenas perturbações na condição inicial produzem pequenas alterações nas aproximações subsequentes, i.e. os resultados dependem continuamente dos dados iniciais.

Exemplo 4.4.2.

Consideramos o seguinte PVI
y′=-40⁢y⁢,0<t≤1, (4.133a)
y⁢(0)=13. (4.133b)

A solução exata é y⁢(t)=15⁢e-40⁢t. Na tabela abaixo, temos os resultados obtidos por computações com o Método de Euler (Explícito, y~e) e o Método de Euler Implícito (y~i) para h=10-1 e 10-2.

h |y~e⁢(1)-y⁢(1)| |y~i⁢(1)-y⁢(1)|
10-1 2.0⁢E+04 3.8⁢E-08
10-2 1.4⁢E-18 8.1⁢E-16

Estabilidade do Euler Explícito

Consideramos o PVI
y′=λ⁢y,t>0, (4.134a)
y⁢(0)=y0, (4.134b)

para dados λ<0 e y0∈ℝ.

A iteração do Método de Euler Explícito para este PVI consiste em

y(0)=y0, (4.135)
y(k+1)=y(k)+h⁢λ⁢y(k),

donde temos

y(k+1)=(1+h⁢λ)k+1⁢y0. (4.136)

Tendo em vista a solução exata y⁢(t)=y0⁢eλ⁢t, temos que o erro global é

|y⁢(t(k))-y(k)| =|y0⁢eλ⁢h⁢k-y0⁢(1+h⁢λ)k| (4.137)
=|(eλ⁢h)k-(1+h⁢λ)k|⁢|y0| (4.138)

e, portanto, a exatidão é determinada por quão bem 1+h⁢λ aproxima eλ⁢h. Observamos que, para qualquer λ<0, (eλ⁢h)k→0 quando t→∞. Por outro lado, para y(k)→0, quando t→0, é necessário que |1+h⁢λ|<1, i.e. o passo do Método de Euler fica restrito

h<2|λ| (4.139)

Supondo um erro de arredondamento δ0 (apenas) na condição inicial, as aproximações subsequentes do Método de Euler Explícito ficariam

y(k+1)=(1+h⁢λ)k+1⁢(y0+δ0), (4.140)

donde temos que

δ(k)=(1+h⁢λ)k⁢δ0 (4.141)

é o valor propagado de δ0 na k-ésima iteração. Ou seja, quando |1+h⁢λ|>1, temos que δ(k)→∞ quando k→∞ e o método é instável. Concluímos que o Método de Euler Explícito é estável para

h<2|λ|. (4.142)

Estabilidade do Euler Implícito

O Método de Euler Implícito é incondicionalmente estável. Para o PVI (4.134), o método produz as aproximações

y(k)=(1-h⁢λ)-k⁢y0. (4.143)

Aqui, para qualquer λ<0, temos que

(1-h⁢λ)-k→0,k→∞, (4.144)

para qualquer escolha do passo h>0. Isto mostra a estabilidade incondicional do método. Também, o Exercício LABEL:cap_pvi_sec:exer:exatidao mostra que o método é convergente para o PVI (4.134).

Exercício 4.4.1.

Considere o seguinte problema de valor inicial

y′=50⁢y+50,0<t≤1, (4.145a)
y⁢(0)=0, (4.145b)

com solução exata y⁢(t)=e50⁢t-1. Use o Método de Euler Implícito com h=10-1 para computar uma aproximação de y⁢(1). Então, verifique a ordem de convergência para diferentes passos h=10-1⁢,10-2⁢,10-3 e 10-4.

Resp.

Dica.

y(k+1)=y(k)+h⁢f⁢(t(k=1),y(k+1)) (4.146)
y(k+1)=y(k)+h⁢(50⁢y(k+1)+50) (4.147)
y(k+1)=y(k)+50⁢h1-50⁢h. (4.148)

Exercício 4.4.7.

Mostre que o Método de Euler Implícito é convergente para a solução exata do PVI (4.134) para qualquer λ<0.

Resp.
y⁢(t(k)) =y0⁢eλ⁢t(k) (4.153)
=y0⁢limh→0+⁡(1-h⁢λ)-t(k)/h (4.154)
=y0⁢limh→0+⁡(1-h⁢λ)-k (4.155)
=limh→0+⁡y(k). (4.156)
