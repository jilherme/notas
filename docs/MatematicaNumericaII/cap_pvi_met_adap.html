4.4 Método adaptativo com controle de erro

Necessita revisão!

Consideremos um problema de valor inicia

y′⁢(t) =f⁢(t,y⁢(t)),t>t0, (4.123)
y⁢(t0) =y0. (4.124)

e um método de passo simples

y(1) =y0, (4.125)
y(i+1)⁢(h(i+1)) =y(i)+h(i+1)⁢Φ⁢(t(i),y(i);h(i+1)), (4.126)

com t(i)=t0+(i-1)⁢h(i). Nesta seção, discutiremos uma estimava para o maior valor de h(i+1) tal que o erro de discretização global e⁢(t(i+1);h(i+1)) seja controlado por uma dada tolerância T⁢O⁢L, i.e.

|e⁢(t(i+1);h(i+1))|:=|y(i+1)⁢(h(i+1))-y⁢(t(i+1))|≈T⁢O⁢L. (4.127)

Para um método de ordem hp, pode-se mostrar que (veja, [3, Cap. 7, Seç. 7.2])

y(i+1)⁢(h(i+1))=y⁢(t(i+1))+ep⁢(t(i+1))⁢(h(i+1))p, (4.128)

onde e⁢(t(i+1)) é uma função apropriada. Então, assumindo que e⁢(t(i);h(i))=0, temos

ep⁢(t(i+1))=h(i+1)⁢ep′⁢(t(i)) (4.129)

e, portanto, para termos (4.127) impomos que

|(h(i+1))p+1⁢ep′⁢(t(i))|=T⁢O⁢L. (4.130)

Daí, se obtermos uma aproximação para ep′⁢(t(i)) teremos uma aproximação para o passo h(i+1).

Para estimarmos ep⁢(t(i+1)), observamos que de (4.128) temos

y(i+1)⁢(h(i+1)2)=y⁢(t(i+1))+ep⁢(t(i+1))⁢(h(i+1))p2p (4.131)

e, então, subtraindo esta de (4.128) temos

y(i+1)⁢(h(i+1))-y(i+1)⁢(h(i+1)2)=ep⁢(t(i+1))⁢(h(i+1)2)p⁢(2p-1), (4.132)

donde

ep⁢(t(i+1))⁢(h(i+1)2)p=y(i+1)⁢(h(i+1))-y(i+1)⁢(h(i+1)2)2p-1. (4.133)

Daí, de (4.129), obtemos

ep′⁢(t(i))⁢h(i+1)⁢(h(i+1)2)p=y(i+1)⁢(h(i+1))-y(i+1)⁢(h(i+1)2)2p-1, (4.134)

o que nos fornece a seguinte aproximação de ep′⁢(t(i))

ep′⁢(t(i))=1(h(i+1))p+1⁢2p2p-1⁢[y(i+1)⁢(h(i+1))-y(i+1)⁢(h(i+1)2)]. (4.135)

Assim sendo, de (4.130) temos que o passo h(i+1) apropriado é tal que

2p2p-1⁢|y(i+1)⁢(h(i+1))-y(i+1)⁢(h(i+1)2)|≈T⁢O⁢L. (4.136)

Com base nesta estimativa podemos propor o seguinte método de passo adaptativo. Partindo de uma escolha arbitrária de h, computamos y(i+1)⁢(h) e y(i+1)⁢(h/2) de y(i). Então, enquanto

2p2p-1⁢|y(i+1)⁢(h)-y(i+1)⁢(h2)|>T⁢O⁢L, (4.137)

tomamos sucessivas divisões de h por 2, até satisfazermos (4.136). Obtido o h que satisfaz (4.136), temos computado y(i+1) com h(i+1)=h.

Exemplo 4.4.1.

Consideremos o seguinte problema de valor inicial

y′-y =sen⁡(t),t>0 (4.138)
y⁢(0) =12. (4.139)

Exercício 4.4.1.

Considere o seguinte problema de valor inicial

y′ +e-y2+1=2,t>1, (4.140)
y⁢(1) =-1. (4.141)

Use o método de Euler com passo adaptativo para computar o valor aproximado de y⁢(2). Para tanto, utilize o passo inicial h=0,1 e a tolerância de T⁢O⁢L=10-4.

Resp.

-5.99240⁢E-1
